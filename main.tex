\section{Minkowski space and space-like surfaces}\label{sc:splsrf}
The environment we will work within is the Minkowki 3-space, defined as $\R^3$ endowed with the pseudometric
\[ds^2 = ( dx^1)^{2} + (dx^2)^{2}-(dx^3)^2.\]
In order to distinguish the use of this metric from the standard euclidean metric of $\R^3$ we will use the notation $\M$. We will also denote the coordinates either as $(x^1,x^2,x^3)$ or $(x,y,t)$, choosing the notation according to convenience and will identify the slice $\{(x,y,0)\}$ of $\M$ with $\R^2$ and with $\C$.\\

The space $\M$ is an example of a more general object called a pseudo-Euclidean vector space, a term that refers to a finite-dimensional vector space $V$ endowed with a nondegenerate (possibly not positive-definite) symmetric bilinear form $g$. This kind of space serves as the ``infinitesimal model'' for pseudo-Riemannian manifolds, smooth manifolds endowed with a smooth nondegenerate 0-2 tensor. If the signature of this tensor is exactly $(n-1,1)$, the manifold is called a Lorentzian manifold.\\
The following definition, borrowed from physics, will be useful later.
\begin{defn*}
	Given $(M,g)$ a Lorentzian manifold and $p \in M$, we say that a vector $v \in T_pM$ is
	\begin{itemize}[noitemsep]
		\item {space-like} if $\langle v,v\rangle > 0$,
		\item {light-like} if $\langle v,v\rangle = 0$ and $v \neq 0$,
		\item {time-like} if $\langle v,v\rangle < 0$.
	\end{itemize}
	We also call a subspace $P \subseteq T_pM$ space-like if every nonzero vector in $P$ is space-like.
\end{defn*}
\paragraph{Space-like immersions.}\mbox{}\\

We will always denote with $S$ a differentiable manifold of dimension  $2$  (a surface) immersed in $\M$, $\iota:S\hookrightarrow \M$. We assume that the ambient pseudometric, $g_{\M}$ when restricted to $S$, yields a nondegenerate positive definite bilinear form. We call this form the first fundamental form or pull-back pseudometric, which we indicate as $\I = \iota^*(g_{\M}) $ and write in coordinates as $\I = g_{ij}dx^i dx^j$ (for simplicity we sometimes forget the immersion $\iota$). By definition, this condition is equivalent to requiring that every tangent plane to $S$ be space-like, and such a surface is called a space-like surface.\\


A nice property of these surfaces is that they are locally the graph of a function of the form $x^3 = f(x^1,x^2)$, $f$ being smooth and Lipschitz with constant less than $1$.\\
We can show this with a simple argument:\\
First, we assumed that the restriction of the Lorentzian metric to each tangent space $T_pS$ is positive-definite so that it is a space-like plane. This can be rewritten as
\[
T_pS \subset \mathscr{C}:=\{v\in \M:\langle v,v\rangle \ge 0\}
=\{t^2\le x^2+y^2\}\ .
\]
The set $\mathscr{C}$ is the outside-region of the double cone in $\M$ whose axis is $\{x^1=0,\ x^2=0\}$. Clearly $\mathscr{C}$ does not contain any vertical vector and clearly the same holds for $T_pM$. 
Therefore we can apply the implicit function theorem and express $S$ locally near $p$ as the graph of a function $t=f(x,y)$, where $f:U\subset \R^2\to \R$.
Where this local representation is valid, we can parametrize locally the surface as $(x,y,f(x,y))$, and obtain the local frame
\[
V_x=(1,0,\partial_x f(x,y)), \qquad V_y=(0,1,\partial_y f(x,y)),
\]
for which the first fundamental form is represented by the matrix
\[
g=
\begin{pmatrix}
\langle V_x,V_x\rangle & \langle V_x,V_y\rangle\\
\langle V_y,V_x\rangle & \langle V_y,V_y\rangle
\end{pmatrix}
=
\begin{pmatrix}
1-f_x^2 & -f_xf_y\\
-f_xf_y & 1-f_y^2
\end{pmatrix}.
\]
Since the parametrization is regular, the condition that $g$ be positive-definite is equivalent to
\[
0<\det g = 1-(f_x^2+f_y^2)=1-|\nabla f|^2,
\]
and therefore $|\nabla f|^2<1$.
\paragraph{Entire surfaces.}\mbox{}\\
Given a surface $S$ immersed in  $\M$, we can define a projection $\Pi:S\to \R^2$ by forgetting the time coordinate, $\Pi(x^1,x^2,x^3):=(x^1,x^2)$ for any $(x^1,x^2,x^3)\in S$. We call the surface $S$ is entire when the map $\Pi$ is a diffeomorphism. If the surface is globally a graph of a function $\varphi$, this condition is exactly stating that the function $\varphi$ is an entire map from $\R^2$ to $\R$.\\

The situation for complete space-like surfaces is actually a lot nicer than what we presented up to now, as the representation as a graph is not only local but actually global as we prove in the proposition below.
\begin{prop}\label{prop:topproprspimm}
	Let $\Omega$ be a domain in $\R^2$ (that is, an open connected set), let $S \overset{\iota}{\hookrightarrow}\M$ be a space-like immersion and $\psi:\Omega\to S$ a parametrization of $S$. Suppose that $S$ is complete with respect to the induced Riemannian metric. Then $S$ is an entire graph and $\Omega$ is simply connected.

\end{prop}
Before laying out the proof let us recall a definition and a proposition from the theory of coverings. Let $M$ and $N$ two differentiable manifolds and $f:M\to N$ a smooth map.
\begin{defn}
	We say that the map $f$ satisfies the path lifting property if for every $p\in N$ and for every path $\alpha: I \to M$ such that $\alpha(0) = f(p)$ there exists a path $\tilde \alpha: I \to N$ such that $\tilde \alpha(0)=p$ and $f\circ \tilde \alpha = \alpha$. We can assume without loss of generality that the paths involved are differentiable.
\end{defn}
\begin{prop}
	If $f$ is a local homeomorphism and satisfies the path lifting property, then it is a covering map.
\end{prop}
\begin{proof}[Proof of Proposition \ref{prop:topproprspimm}]
	The idea of the proof is to show that $\Pi\circ \iota$ is a global diffeomrphism. Clearly this implies that $\Pi$ restricted to $S$ is a diffeomeorphism on $\R^2$ and so that $S$ is an entire graph. From that we obtain also that $\Omega$ is simply connected as $\Pi\circ\iota\circ\psi$ is a diffeomorphism with $\R^2$.\\ 

	Let us observe that the target of $\Pi\circ \iota$ is just $\R^2$, which is simply connected, and the domain is $\Omega$, which is connected since it is a domain. We already proved, by showing that $S$ is locally a graph, that the map $\Pi \circ \iota $ is a local homeomorphism.\\
	Having met these conditions, if we show that $\Pi\circ \iota$ satisfies the path lifting property, we conclude that it is a smooth covering map whose domain is simply connected, hence a diffeomorphism.\\

	Let us fix a point $z\in \Omega$ and a path $\gamma:[0,1] \to \R^2$ of bounded length, $\ell(\gamma) < \infty$, such that $\gamma(0) = (\Pi\circ \iota)(z)$. Our goal is to show that $\gamma$ admits a lift to a path $\gamma^*$ on $\Omega$ starting at $z$.\\
	To this end, let us consider the set of times up to which such a lift exists,
	\[
		\mathcal{J}:= \{\tau\in [0,1]\ |\ \exists \gamma^*: [0,t] \to\Omega \text{ such that } (\Pi\circ\iota)\circ \gamma^* = \gamma \text{ and } \gamma^*(0)=z\},
	\]
	and let us define $\bar \tau := \sup \mathcal{J}$.\\

	Let us observe that $0\in \mathcal{J}$ and that $\mathcal{J}$ is open, since $\Pi \circ\iota$ is a local homeomorphism.

	If we prove that $\bar \tau = 1$, then $\gamma$ admits a lift on the whole interval $[0,1]$ and we have finished. Then let us suppose by contradiction that $\bar \tau<1$ and define $\gamma^* : [0,\bar \tau) \to \Omega$ the lift of $\gamma$ up to $\bar \tau$. This curve exists by definition of $\bar \tau$.\\

	
	Finally, we need to prove that the limit $\lim_{\tau \to \bar \tau} \gamma^*(t) $ exists. If this is true we can then extend $\gamma^*$ up to $\bar \tau$, which means that $\bar \tau\in \mathcal{J}$. Since $\mathcal{J}$ is open, there exists an element $\bar \tau +\epsilon \in \mathcal{J},\ \epsilon>0$. This contradicts the definition of $\bar \tau$ and proves $\bar \tau=1$.\\
	
	To prove that $\lim_{\tau \to \bar \tau} \gamma^*(\tau) $ exists we just need a couple of observations:\\
	First we note that the map $\Pi$ dilates distances, as
			\begin{align*}
				\left<v,v \right>_{\M} &= (v^1)^2+(v^2)^2 - (v^3)^2\\
									   &\leq (v^1) ^2 + (v^2) ^2\\
									   &= \left<\Pi v, \Pi v \right>_{\R^2}.
			\end{align*}
	From this it is clear that $\ell(\gamma^*)\leq \ell(\gamma)$.\\
	Now, if we fix any sequence  $\tau_k\to \bar \tau$ and observe that as $\gamma$ is differentiable, it is derivative is bounded, so $\ell(\gamma|_{[\tau_k,\tau_m]})\leq C |\tau_k-\tau_m|$. By observing that
			\begin{align*}
				d(\gamma^*(\tau_k),\gamma^*(\tau_m)) &\leq \ell(\gamma^*|_{[\tau_k,\tau_m]})\\
													 &\leq \ell(\gamma|_{[\tau_k,\tau_m]})\\
											   &\leq C|\tau_k-\tau_m|
			\end{align*}
			we deduce that also $(\gamma^*(\tau_k))_{k}$ is a Cauchy sequence, hence converging since $S$ is complete and $\Omega$ is to be intended with the induced metric.\\
			By the arbitrariness of the sequence  $(\tau_k)_k$ we deduce the limit $\lim_{\tau\to \bar \tau} \gamma^*(\tau) $ exists.
\end{proof}
In this section we have essentially proved that a complete space-like surface immersed in $\M$ is actually the graph of a smooth function $f$ satisfying $|\nabla f|^2<1$. A partial converse also holds: the graph of a smooth function whose gradient has norm strictly smaller than $1$ is a space-like surface, although it may lack completeness.
\paragraph{Normal vectors to a space-like surface in $\M$}\mbox{}\\
Since the ambient metric $g$ is nondegenerate and each tangent plane $T_pS$ is space-like, the restriction of $g$ to $T_pS$ is positive-definite, hence nondegenerate. It follows that, for every $p\in S$,

\begin{equation*}
	\M = T_p \M = (T_p S)^\perp \oplus T_pS \ .
\end{equation*}
Since $S$ has codimension $1$ in $\M$, the orthogonal complement $(T_pS)^\perp$ is one-dimensional. Therefore it is generated by a vector $N = N_p$ normal to $S$.\\

We can easily show that  $N$ must be a time-like vector, since $T_p S$ is space-like.\\
Let us fix $e_1, e_2 \in T_pM$ two independent vectors. Since $N$ belongs to $(T_pM)^\perp$ we have $\left<e_1,N \right> = \left<e_2,N \right> = 0$. This implies that when we look at the representative matrix of $g_p$ written in the basis $(N,e_1,e_2)$, we have
\begin{equation*}
	[g_p] = 
\begin{pmatrix}
\langle N,N\rangle & 0 & 0\\
0 & \langle e_1,e_1\rangle & \langle e_1,e_2\rangle\\
0 & \langle e_2,e_1\rangle & \langle e_2,e_2\rangle.
\end{pmatrix} =:
\begin{pmatrix}
\langle N, N\rangle & 0 \\
0 & [(g|_S)_p]
\end{pmatrix}.
\end{equation*}
Where the block $[(g|_S)_p]$ represents the form $g$ restricted to the subspace $T_pM$. We conclude that $\left<N,N \right>$ is negative just by observing that $\det [g_p] <0$ as $g$ is a Lorentzian metric, while $\det[(g|_S)_p]>0 $ since  $g$  becomes a metric when restricted to $S$.\\

Since $(T_pS)^\perp$ is $1$-dimensional, all of its vectors are proportional to each other. Moreover, as we have just seen, they are all timelike, so if $w$ is a nonzero vector in $(T_pS)^\perp$, then \[ \langle w,w\rangle<0.\] 
We can thus rescale $w$ by a factor $k\in \R$ such that $\left<kw,kw \right> = -1$. This equation yields two possible choices of $k$, one for each connected component of the space $\hat{\mathbb H}^2=\{v\in \M:\langle v,v\rangle=-1\}$. We can consistently choose the connected component with positive time 
\begin{align*}
	\mathbb H=\mathbb H^2=\{v\in \M:\langle v,v\rangle=-1,\ v^3>0\}
\ \ ,
\end{align*}
effectively selecting a unique normal vector for each $p\in S$. The choice of the component within  $\{v^3>0\} $ or $\{v^3<0\} $ is sometimes referred to a ``time-orientation'' for the space $\M$, and can be performed independently from the surface $S$. 
This way we obtain a globally defined map
\begin{align}\label{eq:gausshmap}
	\tilde  u:S\to \mathbb{H}
\ \ .
\end{align}
\paragraph{$\mathbb{H}$ is a model for the Hyperbolic plane.}\label{sc:Hmodel}\mbox{}\\
The space $\mathbb{H}$ that we just defined is clearly a surface immersed in $\M$, but there is more.
First of all, we observe that, by manipulating the definition of $\mathbb{H}$, we can write it as the graph of the function $\varphi(x,y) = \sqrt{x^2+y^2+1} $. The gradient of such function is $\nabla \varphi = \frac{1 }{\sqrt{x^2+y^2+1} }(x,y) $, from which it is clear that $|\nabla \varphi| <1$. By applying the result we proved before, we obtain that the bilinear form that $\mathbb{H}$ inherited from the immersion in $\M$ is a Riemannian metric.
In fact this space is one of the many models of the hyperbolic plane.\\
Another model for the same Riemannian manifold is the Poincaré disk. As a differentiable manifold, the Poincaré disk is simply the unit ball in $\R^2$, $\mathbb{D}=\{\xi\in \R^2\mid |\xi|\leq 1\} $, while we can construct the metric by exhibiting a diffeomorphism between $\mathbb{D}$ and $\mathbb{H}$ pulling back the metric.\\
A possible diffeomorphism is the map $p$, whose inverse is the map $q$, as can be verified by direct computation:
\begin{align*}
\begin{aligned}
	p\colon \mathbb{H} &\longrightarrow \mathbb{D} \\
	(x,y,t) &\longmapsto \frac{1}{1+t}(x,y)
\end{aligned}
\qquad\qquad
\begin{aligned}
	q\colon \mathbb{D} &\longrightarrow \mathbb{H} \\
	\xi &\longmapsto \frac{1}{1-|\xi|^2}(2\xi^1,2\xi^2,1+|\xi|^2).
\end{aligned}
\end{align*}
It is clear that both functions are smooth, since $|\xi| <1$ and $t>1$.\\


We would like to find the expression of  $q^*(g_\mathbb{H})$.To simplify the calculations we might observe that the metric of $\mathbb{H}$ inherited by $\M$ is invariant under a rotation on the $t$ axis, and that such a rotation corresponds under $p$ to a rotation of the disk. This means that the metric $q^*(g_{H})$ will be invariant under rotation.\\
It is then sufficient to fix a certain $\xi\in \mathbb{D}$ of the form $\xi = (a,0) $ and calculate $q^*(g_\mathbb{H})_\xi$; this will specify the whole metric tensor. The matrix associated to $dq_\xi$ in the canonical bases $\mathscr{B}_1	= (e_1,e_2)$ of $\R^2$ and of $\mathscr{B}_{2}=(E_1,E_2,E_3)$ of $\M$ is
\begin{align*}
	[dq_\xi]_{\mathscr{B}_1 \mathscr{B}_2}
	=
	\frac{2}{(1-|\xi|^2)^2}
	\begin{pmatrix}
		1+(\xi^1)^2-(\xi^2)^2 & 2\xi^1\xi^2 \\
		2\xi^1\xi^2 & 1-(\xi^1)^2+(\xi^2)^2 \\
		2\xi^1 & 2\xi^2
	\end{pmatrix}.
\end{align*}
Which, evaluated at $\xi=(a,0)$, yields
\begin{align*}
	[dq_{(a,0)}]_{\mathscr{B}_1 \mathscr{B}_2}
	=
	\frac{2}{1-a^2}
	\begin{pmatrix}
		\frac{1+a^2}{1-a^2} & 0 \\
		0 & 1 \\
		\frac{2a}{1-a^2} & 0
	\end{pmatrix} =: 
	\frac{2}{1-a^2}\begin{pmatrix} v_1|v_2 \end{pmatrix} .
\end{align*}
We observe that, with respect to the metric of $\M$, it holds
\begin{align*}
	\langle v_1,v_1\rangle = 1,
	\hspace{5mm}
	\langle v_2,v_2\rangle = 1,
	\hspace{5mm}
	\langle v_1,v_2\rangle = 0.
\end{align*}
Therefore $\mathscr{B}_3 = (v_1,v_2)$ is an orthonormal basis of $T_{q(0,a)}\mathbb{H}$ with respect to $g_{\mathbb{H}}$, hence $[g_{\mathbb{H}}]_{\mathscr{B}_3 }= I $, while $dq_{(a,0)}$ represented by
\[[dq_{(a,0)}]_{\mathscr{B}_3\mathscr{B}_1} = \frac{2}{1-a^2}I.\] 
So we have
\begin{align*}
	[q^*(g_{\mathbb H})_{(0,a)}]_{\mathscr{B}_1}
	&=
	[dq_{(0,a)}]_{\mathscr{B}_3 \mathscr{B}_1}^T[g_{\mathbb H}]_{\mathscr{B}_3}[dq_{(0,a)}]_{\mathscr{B}_3 \mathscr{B}_1}\\
	&=
	[dq_{(0,a)}]_{\mathscr{B}_3 \mathscr{B}_1}^TI[dq_{(0,a)}]_{\mathscr{B}_3 \mathscr{B}_1}\\
	&=
	\left(\frac{2}{1-a^2}\right)^2 I
\ \ .
\end{align*}
In particular, $q^*(g_{\mathbb H})_{(0,a)}$ is a multiple of the Euclidean metric.\\
By the argument on rotational invariance, we conclude that for every $\xi\in \mathbb{D}$,
\begin{align*}
	q^*(g_\mathbb{H})_\xi
	=
	\left(\frac{2}{1-|\xi|^2}\right)^2 g^E_\xi.
\end{align*}
The disk $\mathbb{D}$ with this metric, which we denote also with $ds_p^2$, is called the Poincaré model of the Hyperbolic plane.\\
\paragraph{The Gauss Map in $\M$}\mbox{}\\
In \eqref{eq:gausshmap} we constructed a map $\tilde u$ between the surface $S$ and $\mathbb{H}$. Having now at our disposal the isometry $q$ between $\mathbb{H}$ and the Poincaré disk, $\mathbb{D}$, we can compose $\tilde u$ with $q$, obtaining what is called the Gauss map of the surface $S$. 
\[ u := q \circ \tilde u:S\to \mathbb{D}. \]
The construction we followed is analogous to the classical one, except that the target of the Gauss map is here $\mathbb{D}^2$ instead of $\mathbb{S}^2$. Moreover, unlike the Euclidean case, for a space-like surface we always have a globally defined unit normal field: once a time orientation of $\M$ is fixed, the timelike line $(T_pS)^\perp$ intersects the corresponding connected component of $\hat{\mathbb{H}}^2$ in exactly one point for every $p\in S$. Instead, for an immersed surface in Euclidean space, there is no canonical way to choose a normal vector.\\

\section{The Gauss-Codazzi equations and the curvatures of $S$}
In this section we derive the Gauss-Codazzi equations for a space-like surface immersed in $\M$ and see that they assume a different form from the Euclidean case. These equations will play a central role in the discussion that follows, especially due to Bonnet's fundamental theorem on surfaces, stated at the end of the section.
\subsection{Recall on the second fundamental form}
We repeat now the construction of the second fundamental form of the immersion $\iota$, which brings a different sign convention that will occur also to the expression of the Gaussian curvature.\\


As we have already noted, given a point $p\in S$, since $S$ is a space-like surface, we can decompose the tangent space
\[T_p \M = T_p S \oplus (T_pS)^\perp\ . \] 
This splitting is differentiable, in the sense that the mappings
 \[(p,v) \mapsto (p,v^T) \qquad\text{and}\qquad (p,v)\mapsto (p,v^N)  \] 
 are differentiable maps of $T\M|_S$ to $T\M|_S$.

Denote with $D$ the Levi-Civita connection on $\M$ induced by the pseudometric $g_{\M}$ and with $\nabla$ the Levi-Civita connection of $(S,\I)$. As usual, if $X$ and $Y$ are local vector fields on $S$ with local extensions $\bar X$ and $\bar Y$ then the induced connection on  $S$ is simply the projection of the connection of the ambient space onto the surface $S$
\[ \nabla_X Y = \left( D_{\bar X} \bar Y \right)^T\ .\] 
Which means that the difference between the two connections of $X$ and $Y$ will be a local vector field on $\M$  normal to $S$,
\begin{align}\label{eq:defB}
	B(X,Y)_p := (D_{\bar X} \bar Y - \nabla_X Y)_p \perp T_pS.
\ \ . 
\end{align}
As in the Euclidean case, $B$ is a symmetric tensor and is independent of the chosen extensions $\overline X$ and $\overline Y$. Since the normal space $(T_pS)^\perp$ is one-dimensional, once a local unit normal vector field $N$ along $S$ is fixed, there exists a unique symmetric bilinear form $\II = \II(X,Y)$ such that
\[
B(X,Y)_p=(\II(X,Y)N)_p,
\]

From now on we forget to distinguish a vector field $X$ with its extension $\overline{X}$. Let $N$ be local normal vector field on $S$ and $V,W$ two local tangent vector fields to $S$. We observe first that
\[\left<N,W \right> \equiv 0\ ,\] 
so that by taking the derivative along $V$,
\begin{align*}
	\left<D_V N,W \right> + \left<N,D_VW \right> = 0
	\ \ .
\end{align*}
Then
\begin{align}\label{eq:gc7}
	\left<D_VN,W \right> &= -\left<N,B(V,W)+\nabla_VW \right>\notag\\
	&= -\left<N,B(V,W) \right> \notag\\
	&= -\left<N,\II(V,W)N \right> \notag\\
	&= -\left<N,N \right>\II(V,W) \notag\\
	&=  \II(V,W)
\ \ . 
\end{align}
With this choice we have, from definition \ref{eq:defB},
\begin{align}\label{eq:gc1}
	\boxed{D_V X= \nabla_V X + \II(V,X)N }
\ \ . 
\end{align}
Now, having fixed $W_p\in T_pS$, and since the metric on $T_pS$ is a nondegenerate bilinear form, we can associate to the functional $\II(\cdot,W_p):T_pS\to \mathbb{R}$ a unique vector $\II(\cdot,W_p)^\sharp\in T_pS$ such that for every $V_p\in T_pS$,
\[
\left<\II(\cdot,W_p)^\sharp,V_p\right>=\II(V_p,W_p)\ .
\]
We then define the shape operator by
\[
A_pW_p:=\II(\cdot,W_p)^\sharp.
\]
Thus
\begin{align*}
\left<A_pW_p,V_p\right>
&=\II(V_p,W_p)\\
&=\II(W_p,V_p)\\
	\commb{\qref{eq:gc7}}\qquad &=\left<D_{W_p}N,V_p\right>.
\end{align*}
Since this holds for every $V_p\in T_pS$, we obtain
\[
A_pW_p=D_{W_p}N.\ 
\]
So we have $A = DN$.
\subsection{The Gauss-Codazzi equations and Gaussian curvature}
It is easy to see that $\M$ is actually flat with its metric. This mean that its curvature tensor is zero $\overline{R} (X,Y)Z \equiv 0 $.
Let us rewrite $\overline{R}$ in relation with $R$ the curvature tensor of the surface $S$.
By definition
\begin{align*}
	\overline{R}(X,Y) Z = D_X D_Y Z - D_Y D_X Z - D_{[X,Y]}Z
\ \ . 
\end{align*}
Let us focus first on the term $D_X D_Y Z $, applying the equation \eqref{eq:gc1} we get
\begin{align*}
	D_X D_Y Z &= D_X (\nabla_Y Z +\II(Z,Y)N)\\
	&= \nabla_X \nabla_Y Z + \II\left(\nabla_Y,X\right) N + d_X(\II(Z,Y))N + \II(Z,Y)D_XN
\ \ . 
\end{align*}
And analogously
\begin{align*}
	D_Y D_X Z &= \nabla_Y \nabla_X Z + \II\left(\nabla_X,Y\right) N + d_Y(\II(Z,X))N + \II(Z,X)D_YN
\ \ . 
\end{align*}
Finally we have, by noting that the connection $\nabla$ is symmetric,
\begin{align*}
	D_{[X,Y]}Z &= \nabla_{[X,Y]}Z - \II([X,Y],Z)N\\
	&=   \nabla_{[X,Y]}Z + \II(Z,\nabla_X Y)N - \II(Z,\nabla_Y X)N
\ \ . 
\end{align*}
So that, by putting everything together we obtain,
\begin{align}\label{eq:gc2}
	0 &= \overline{R}(X,Y)Z \notag\\
	  &= D_X D_Y Z - D_Y D_X Z- D_{[X,Y]}Z\notag\\ % &=\numunderarrow{\astr{4}}{D_X D_Y Z - D_Y D_X Z} - D_{[X,Y]}Z\\
	&= \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]}Z\notag\\
	& \qquad\qquad+ d_X(\II(Z,Y))N-\II (\nabla_X Z, Y) N -\II(Z,\nabla_X Y)N \notag\\
	& \qquad \hspace{5.75mm}
	-\left[ d_Y(\II(Z,X))N-\II (\nabla_YZ,X)N - \II(Z,\nabla_Y X)N\right]\notag\\
	& \qquad \qquad +\II(Z,Y)D_XN-\II(Z,X)D_YN
\ \ . 
\end{align}
Now, we recognize in the expression that $R(X,Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]}Z$, and by definition, we set the derivative of the tensor,\[(\nabla_X(\II))(Y,Z) = d_X(\II(Z,Y))-\II (\nabla_X Z, Y)  -\II(Z,\nabla_X Y)\ .\] So, by continuing in \eqref{eq:gc2},
\begin{multline}\label{eq:gc3}
	0 = R(X,Y)Z + (\nabla_X \II)(Y,Z)N - (\nabla_Y\II) (Z,X)N\\ +\II(Z,Y)D_XN-\II(Z,X)D_YN
\ \ . 
\end{multline}
Now we observe that from  $\left<N,N \right> \equiv -1$ we have, for $v$ tangent to $S$, 
\[2\left<D_vN,N \right> = 0\] 
so that $D_vN$ is a tangent vector on $S$.
Now, in \eqref{eq:gc3} we can separate the terms that are tangent to $S$ and those that are normal. This way we obtain two distinct equations
\begin{align}\label{eq:gc4}
	\begin{cases}
		R(X,Y)Z  = -\II(Z,Y)D_XN+\II(Z,X)D_YN\\
		(\nabla_X \II)(Y,Z) - (\nabla_Y\II) (Z,X) = 0
	\end{cases}
\ \ . 
\end{align}
These equations are (a form of) the Gauss-Codazzi equations.

Let us take the first equation of \eqref{eq:gc3} and test it by a tangent local vector field $W$. We get
\begin{align*}
	\left<R(X,Y)Z,W \right> &=  \left<- \II(Z,Y)D_XN+\II(Z,X)D_YN, W\right>\\
							&= -\II(Z,Y)\left< D_XN, W\right> +\II(Z,X)\left<D_YN,W \right>\\
							&=-\II(Z,Y)\II(X,W) +\II(Z,X)\II(Y,W)
\ \ . 
\end{align*}
Now, if we fix $v_1,v_2$ a basis of $T_pS$ and evaluate the last equation by setting $X=v_1, Y=v_2, Z=v_2, W=v_1$ and $h_{ij}= \II(v_i, v_j)$ we get

\begin{align}\label{eq:gc5}
	\left<R(v_1,v_2)v_2,v_1 \right>	&= -\II(v_2,v_2)\II(v_1,v_1) +\II(v_2,v_1)\II(v_2,v_1)\notag\\
									&= - \left( \II(v_2,v_2)\II(v_1,v_1)-\II(v_1,v_2)^2 \right)\notag\\
									&= -\det(h_{ij}) 
\ \ .
\end{align}
Now, by definition of sectional curvature and setting $g_{ij} =\I( v_i,v_j) $, 
\begin{align}\label{eq:gc6}
	\left<R(v_1,v_2)v_2,v_1 \right> = K(v_1,v_2)\det(g_{ij}) 
\ \ . 
\end{align}
Here $K(v_1,v_2)$ is the sectional curvature of the plane spanned by $v_1,v_2$ which is simply the gaussian curvature $K$ since we're in dimension $2$.\\
So by putting together \eqref{eq:gc5} and  \eqref{eq:gc6},
\begin{align*}
	K = -\det_\I (\II) = - \frac{\det(h_{ij})}{\det(g_{ij})}
\ \ . 
\end{align*}
Which is the same as the Gauss equation for surfaces in Euclidean space if not for a negative sign.\\
Let us also focus on the other equation. We can define the exterior derivative of a tensor with respect to the connection, \[ d^\nabla T(X,Y) = \nabla_X(TY)-\nabla_Y(TX) -T[X,Y]  = (\nabla_X T)(Y) - (\nabla_Y T)(X)\ .\] This way, Codazzi's equation becomes
\[d^\nabla \II = 0.\] 
So that the two equations get the following form:
\begin{equation}
\begin{cases}\label{eq:gc}
	\displaystyle \det_\I (\II)= -K  \\
	\displaystyle d^\nabla \II = 0. 
\end{cases}	
\end{equation}
\paragraph{Bonnet's theorem}\mbox{}\\
The Gauss-Codazzi equations play a fundamental role when the ambient manifold has constant sectional curvature. In this case, they are not only necessary compatibility conditions for the existence of an isometric immersion, but, on a simply connected surface, they are also sufficient.
\begin{thm}[Bonnet]\label{th:bonnet}
	Let $(g_{ij})_{ij}$	and $(h_{ij})_{ij}$ be differentiable functions defined on an open set $V\subset \R^2$ to $\R$ such that $g = (g_{ij})_{ij}$ is is a symmetric positive definite matrix and $h = (h_{ij})_{ij}$ is symmetric. Suppose that they satisfy the equation \eqref{eq:gc}.\\
	Then for every $q\in V$ there exists a neighborhood $U\subset V$ of $q$ and a diffeomorphism $\mathbf{x}:U\to \mathbf{x}(U) \subset \M$ such that the regular surface $\mathbf{x}(U)$ has $g$ and $h$ as first and second fundamental forms respectively.\\
	Furthermore, if $U$ is connected and if 
	\[\overline{\mathbf{x}} :U\to  \overline{\mathbf{x}}(U) \subset \M\] 
	 is another diffeomorphism satisfying the same conditions, then there exist an isometry $T:\M\to \M$ such that $\overline{\mathbf{x}} = T \circ \mathbf{x}$.
\end{thm}
We redirect to \cite{do_carmo_2016} for a proof of the Euclidean case, whose only difference is the sign of Gauss' equation.
\paragraph{Mean curvature}\mbox{}\\
We conclude this section by briefly recalling the definition of mean curvature, which does not differ from the usual Euclidean setting.
As we have already seen, we can associate to the bilinear form $\II$ a linear operator $A_p:T_pS\to T_pS$, called shape operator, such that $\II(v,w)=\left<A_p(v),w\right>$.\\
We define the mean curvature of $S$ at $p$ as
\begin{align*}
	H(p)=\frac{1}{2}\tr(A_p)
	\ \ .
\end{align*}
If we again fix a basis $v_1,v_2$, denote with $(g_{ij})_{ij}$ and $(h_{ij})_{ij}$ the coefficients of the first and second fundamental forms, and express $A_p$ as $A_p(v_i)=A_i^{\ k}v_k$, we get
\begin{align*}
	h_{ij}
	=\II(v_i,v_j)
	=\left<A_p(v_i),v_j\right>
	=A_i^{\ k}g_{kj}
	\ \ .
\end{align*}
If $(g^{ij})$ is the inverse matrix of $(g_{ij})$, it follows that
\begin{align*}
	A_i^{\ k}=h_{ij}g^{jk},
\end{align*}
and therefore
\begin{align*}
	\tr(A_p)=A_i^{\ i}=h_{ij}g^{ji}=g^{ij}h_{ij}
	\ \ .
\end{align*}
So that we have
\begin{align*}
	H=\frac{1}{2}\tr(A_p)=\frac{1}{2}A^{\ i}_i	= \frac{1}{2} g^{ij}h_{ij}=\frac{1}{2}\tr_{\I}(\II)
	\ \ .
\end{align*}
Now, since
\begin{align*}
	(g^{ij})
	=
	\frac{1}{\det(g_{ij})}
	\begin{pmatrix}
		g_{22} & -g_{12}\\
		-g_{12} & g_{11}
	\end{pmatrix}
	\ \ ,
\end{align*}
we obtain
\begin{align*}
	2H
	=
	g^{ij}h_{ij}
	=
	\frac{h_{22}g_{11}-2h_{12}g_{12}+h_{11}g_{22}}{\det(g_{ij})}
	\ \ .
\end{align*}
As we already mentioned, there is no formal difference with surfaces immersed in $\R^3$.
\section{Riemann surfaces and harmonic maps}
We here introduce some notation and some fundamental concepts of Riemann surfaces. The most crucial part will be the fact that each metric on a surface is locally conformal to the Euclidean, the relationship between a Riemannian metric and a complex structure, the Uniformization theorem and the definition of the Hopf differential.

\subsection{Preliminaries on Riemann surfaces}

Let us fix $S$ a surface (a topological manifold of dimension 2).
\begin{defn}
	A complex atlas $\mathcal{C}$ is a collection of charts 
	 \[\mathcal{C} = \{z_{\alpha}: U_{\alpha} \to  \C \} \] 
	 that covers the whole surface $S$ and whose transition maps
	\begin{align*}
	z_{\beta}\circ z_{\alpha}^{-1} &: \; z_{\alpha}\big(U_{\alpha}\cap U_{\beta}\big) \to z_{\beta}\big(U_{\alpha}\cap U_{\beta}\big)
	\end{align*}
	 are holomorphic as maps between open subsets of $\C$. We call $\mathscr{C}$ a complex or conformal structure if it is an atlas maximal under the subset relation. Finally we call the pair $(S,\mathscr{C})$ (just $S$ for short) a Riemann surface.
\end{defn}
\begin{defn}
	A continuous map $h: S_1 \to S_2$ between Riemann surfaces is said to be holomorphic if, in local coordinates  $\{U_\alpha,z_\alpha\}$ on $S_1$ and $\{U'_\beta,z'_\beta\}$ on $S_2$,all the maps $z'_\beta\circ h\circ z_\alpha^{-1}$ are holomorphic wherever they are defined.
	We call $f$ a biholomorphism if it is holomorphic, bijective and its inverse  $f^{-1}$ is holomoprhic.\\
	We say that the two Riemann surfaces $S_1$ and $S_2$ are isomorphic if there exists a biholomoprhism between them.
\end{defn}

\begin{defn}
	We say that two metrics $g$ and $g'$ on a surface $S$ are conformal to each other if there exists a smooth function $\upsilon:S\to \R$ such that
	\begin{align*}
		g' = e^{2\upsilon} g
		\ \ .
	\end{align*}
	Equivalently, $g'$ is at each point a positive multiple of $g$ by a smooth function. The function $e^{2\upsilon}$ is called the conformal factor.
\end{defn}
\paragraph{Conformal structures on 2-surfaces}\mbox{}\\
Here we state the results that lead to the fact that every Riemannian surface admits a complex structure.
\begin{thm}
	Given  a Riemannian metric $g$ on a differentiable surface $S$, and a point $p\in S$, there is a chart near $p$, $\varphi:U\to \R^2$ such that $g$ takes the form and a smooth positive function $\lambda$ such that
	\[g  = \lambda^2(dx^2+dy^2) = e^{2v}(dx^2+dy^2),\]
	where $(x,y)$ are the coordinates defined by $\varphi$.
\end{thm}
\begin{defn}
	The chart $(\varphi,U)$ in the theorem is called conformal, while the coordinates $(x,y)$ on $U$ are called isothermal coordinates around $p$.\\
\end{defn}
Given a conformal chart we may introduce the change of coordinates
\begin{align*}
	\begin{cases}
		z = x+iy\\
		\bar z = x-iy
	\end{cases}
	\quad \text{or equivalently}
	\quad
	\begin{cases}
		x = \frac{1}{2} (z + \bar z)\\
		y = \frac{i}{2}(z- \bar z)
	\end{cases}
\ \ . 
\end{align*}
The associated coordinated fields and forms can be expressed as
\begin{align*}
	\begin{cases}
		\displaystyle\frac{\partial  }{\partial z}  = \frac{1}{2}( \frac{\partial  }{\partial x} - i \frac{\partial  }{\partial y} ) \\
		\displaystyle\frac{\partial  }{\partial \bar z} = \frac{1}{2} ( \frac{\partial  }{\partial x} + i \frac{\partial  }{\partial y} ) 
	\end{cases}
	\qquad \text{and}
	\qquad
	\begin{cases}
		dz = dx + idy \\
		d \bar z = dx -idy
	\end{cases}
\ \ . 
\end{align*}
This lets us write the Euclidean metric as
\begin{align*}
	dx^2+dy^2 &=  dx \otimes dx + dy \otimes dy \\
		&= \frac{1}{4} d(z+ \bar z) \otimes d(z + \bar z) - \frac{1}{4} d(z-\bar z) \otimes d(z- \bar z)\\
		&= \frac{1}{2}( dz \otimes d \bar z + d \bar z \otimes dz)\\
		&= \Re(dz \otimes d \bar z)\\
		&=  dz d \bar z\\
		&= |dz|^2
\ \ , 
\end{align*}
where the last two equalities are definitions.\\
So we can express the metric $g$ as
\[ g = e^{2v} |dz|^2.\] 

The change of coordinates $(x,y)\mapsto z=x+iy$ yields charts with values in $\C$. Using these charts, we can endow $S$ with a complex structure.
\begin{prop}
	Fix an orientation on $S$. Given a conformal chart $(\varphi,U)$ preserving the orientation, denote by $(\tilde\varphi,U)$ the associated complex chart. Then
	\begin{align*}
		\{(\tilde\varphi,U)\mid (\varphi,U)\text{ is an orientation-preserving conformal chart on }(S,g)\}
	\end{align*}
	constitutes a complex structure on $S$.
\end{prop}
Now, let us recall some operators that take a simple form when read through a conformal chart, for which the metric has the form $e^{2u}|dz|^2$. The proof of these formulas can be found in \cite{jost2006compact}.\\
First, the Laplacian with respect to the metric $g$ is
\[\Delta_g = \frac{1}{\lambda^2}(\frac{\partial  }{\partial x^2} +\frac{\partial }{\partial y^2} ) = e^{-2u}(\frac{\partial  }{\partial x^2} + \frac{\partial  }{\partial y^2} ) = e^{-2u}\frac{\partial  }{\partial z} \frac{\partial  }{\partial \overline{z}} .\] 
While the Gauss curvature of the surface takes the form
\[K_g = -\Delta_g\log(\lambda) = -2\Delta u.\]
Let us finally observe that,
\begin{align*}
	dz^2 &= dz\otimes dz\\
		&= d(x+iy) \otimes d(x+iy)\\
		&= dx \otimes dx +i \left( dx \otimes dy + dy \otimes dx \right) - dy \otimes dy\\
		&=: dx^2 - dy^2 +2i dxdy
\ ,
\end{align*}
where we define $dxdy = \frac{1}{2} (dx\otimes dy+dy \otimes dx) $.
\paragraph{Integration through a conformal chart}\mbox{}\\
Given a curve $\gamma$ in S, its length is defined by
\[ \ell(\gamma) = \int_\gamma \lambda(z) |dz|\ .\] 
On the other hand the area form can be expressed as follows
\[ \lambda^2 dx\wedge dy = -\frac{i\lambda^2}{2} dz\wedge d\bar z\ ,\] so that the area of a measurable set becomes
\begin{align*}
	\mathrm{Area}(B):=\int_{B}\lambda^{2}(z)\frac{i}{2}dz\wedge d\bar{z}\\
	= \int_{B}\lambda^{2}(z)\frac{i}{2}dzd\bar{z}
\ \ . 
\end{align*}	


\subsection{Uniformization}
An important property of Riemann surfaces is that they can be classified in just three categories, depending on their universal covering. This result is  (a direct consequence of) the uniformization theorem.
In what follows we denote by $\widehat{\C}$ the 2-sphere, with the metric and conformal structure induced by the Euclidean metric of $\R^3$.
\begin{thm}[Uniformization Theorem]
	Let $\Sigma$ be a simply connected Riemann surface. Then $\Sigma$ is isomorphic to exactly one of the following three surfaces:
	\begin{enumerate}
		\item The Riemann sphere $\widehat{\C}$.
		\item The complex plane $\C$.
		\item The Poincaré disk $\mathbb{D}$.
	\end{enumerate} 
\end{thm}
Every Riemann surface $\Sigma$ admits a universal covering $\tilde \Sigma$, which inherits the complex structure of its base space. Since the universal covering $\tilde \Sigma$ of a surface $\Sigma$ is simply connected, we can apply the uniformization theorem to it, making $\Sigma$ fall in one of the following categories. We say that $\Sigma$ is
\begin{itemize}[noitemsep]
	\item elliptic if $\widetilde \Sigma \cong \widehat{\C}$,
	\item parabolic if $\widetilde \Sigma \cong \C$,
	\item hyperbolic if $\widetilde \Sigma \cong \mathbb{D}$.
\end{itemize}
In our case the surface we consider is simply connected and noncompact (for instance because it is the graph of an entire function on $\R^2$) so we can exclude the elliptic case.  
\begin{cor}
	If $\Sigma$ is a simply connected noncompact Riemann surface, then $\Sigma$ is conformally equivalent either to $\C$ or to $\mathbb{D}$.
\end{cor}

\subsection{Harmonic maps into $\mathbb{H}$}
An important result we need to prove is that the Gauss map of a complete space-like surface in $\M$ is harmonic if and only if the surface is CMC. In order to do this we need first to give a definition of harmonicity for a map with values in $\M$. \\
Let us consider $f:(\mathbb{D},e^{2\eta}(dx^2+dy^2))\to  \M$. We can define the energy density of $f$ as 
\[e(f) = e^{-2\eta}(f_x\cdot f_x+f_y\cdot f_y)\ .\] 
If the map has compact support we can integrate the energy density, obtaining the energy of the map,
\begin{align*}
	E(f) &= \int_{\mathbb{D}} e^{2\eta} e(f) dx \wedge dy\\
	&= \int_{\mathbb{D}} f_x\cdot f_x +f_y\cdot f_y dx  \wedge dy
\ \ . 
\end{align*}
This quantity does not define an $\mathbb{R}$-valued functional on the space of all maps $f$, since the integral may be infinite. Nevertheless, if we consider a compactly supported deformation of $f$, the corresponding first variation is still well defined. This in turn lets us give the definition of a harmonic map.\\

Let us fix a compactly supported deformation of $f$, namely a smooth map
\[
\xi:\mathbb{D}\times (-\epsilon,\epsilon) \to \M
\]
such that
\[
\xi(x,y,0) = f(x,y)
\]
for every $(x,y)\in\mathbb{D}$, and there exists a compact set $K\subset \mathbb{D}$ such that
\[
\xi(x,y,t)=f(x,y)
\]
for every $(x,y)\in \mathbb{D}\setminus K$ and every $t\in (-\epsilon,\epsilon)$. We call $\xi$ of this form an admissible deformation of $f$. \\
Then, we define \[ f_t := \xi(\cdot,\cdot,t)\ .\]
\begin{defn}
	We say that $f$ is a harmonic map if for every admissible deformation $\xi$, and for one, equivalently any, compact domain $\Omega\subset \mathbb{D}$ containing the support of the deformation, one has
	\[
	\lim_{t\to 0}\frac{\Delta E_\Omega(t)}{t}=0,
	\]
	where
	\[
	\Delta E_\Omega(t):=\int_\Omega e^{2\eta}\bigl(e(f_t)-e(f)\bigr)\,dx\wedge dy\ .
	\]
\end{defn}
The following lemma is needed to give later a characterizaion of harmonic maps.
\begin{lemma}\label{lm:deren1}
	\[\left.\frac{dE(f_t)}{dt}\right|_{t=0}  = -2\int_{\mathbb{D}}\xi_t\cdot (f_{xx}+f_{yy})dx\wedge dy\ .\] 
\end{lemma}
\begin{proof}
	First we observe that
	\begin{align}\label{eq:deren1}
		\left.\frac{dE(f_t)}{dt}\right|_{t=0}
		&= \left.\int_{\mathbb{D}} \frac{\partial}{\partial t}\bigl(\xi_x\cdot \xi_x+\xi_y\cdot \xi_y\bigr)\right|_{t=0} dx\wedge dy \notag\\
		&= 2\int_{\mathbb{D}} \bigl(\xi_x\cdot \xi_{xt}+\xi_y\cdot \xi_{yt}\bigr)\big|_{t=0}\, dx\wedge dy
		\ \ .
	\end{align}
	Now,
	\begin{align*}
		d(\xi_t\cdot \xi_x\, dy)
		&= (\xi_t\cdot \xi_x)_x\, dx\wedge dy\\
		&= (\xi_{xt}\cdot \xi_x+\xi_t\cdot \xi_{xx})\, dx\wedge dy,
	\end{align*}
	so that
	\begin{align*}
		\xi_x\cdot \xi_{xt}\, dx\wedge dy &=
		d(\xi_t\cdot \xi_x\, dy)-\xi_t\cdot \xi_{xx}\, dx\wedge dy 
		\shortintertext{and similarly}
		\xi_y\cdot \xi_{yt}\, dx\wedge dy &=	-d(\xi_t\cdot \xi_y\, dx)-\xi_t\cdot \xi_{yy}\, dx\wedge dy
	\ . 
	\end{align*}
	Now, by substituting into \eqref{eq:deren1}, we obtain
	\begin{align*}
		\left.\frac{dE(f_t)}{dt}\right|_{t=0}
		&=
		-2\int_{\mathbb{D}} \xi_t\cdot (\xi_{xx}+\xi_{yy})\big|_{t=0}\, dx\wedge dy\\
		&\qquad
		+2\int_{\mathbb{D}} d\bigl(\xi_t\cdot \xi_x\, dy-\xi_t\cdot \xi_y\, dx\bigr)\big|_{t=0}
		\ \ .
	\end{align*}
	Then, since $\xi_t$ has compact support $K$, by applying Stokes' theorem we obtain that the second term of the sum in the last equation is zero. Once recognized that $\xi_{xx}|_{t=0} = f_{xx}$ and $\xi_{yy}|_{t=0}=f_{yy}$ we have proven the lemma.
\end{proof}
\begin{cor}\label{cor:tglapnull}
	Let $f:\mathbb{D}\to \mathbb{H}\subset \M$ a smooth map. Then $f$ is harmonic if and only if the tangent component of its Laplacian is null:
	 \[(\Delta f )^T = 0\ .\] 
	 That is, \[\Delta f+\left<\Delta f,f \right> f = 0\ .\] 
\end{cor}
\begin{proof}
Let us fix $\xi:\mathbb{D}\times (-\epsilon,\epsilon) \to \mathbb{H}$ a compactly supported deformation of $f$ (that is, a deformation of $f$ as above).
By lemma \ref{lm:deren1},
\[\left.\frac{dE(f_t)}{dt}\right|_{t=0} = -2 \int_{\mathbb{D}} \xi_t|_{t=0} \cdot \Delta f \ .\] 
Being $\xi$ a deformation of $f$, which is a smooth map with values in $\mathbb{H}$, we have constantly $\xi(x,y,t) \in \mathbb{H}$, hence $\xi_t(x,y)|_{t=0} \in T_{\xi(x,y,0)}\mathbb{H}^2$. So we obtain 
\begin{align}\label{eq:tglaplnull1}
	\left.\frac{dE(f_t)}{dt}\right|_{t=0} 
		= -2 \int_{\mathbb{D}} \xi_t \cdot  (\Delta f)^T
\ \ . 
\end{align}

Now, one implication is trivial, if $(\Delta f)^T = 0$, then also the LHS is zero for every choice of $\xi$.\\

On the other hand, let us assume that \[\left.\frac{dE(f_t)}{dt}\right|_{t=0}=0 \] for every admissible deformation and prove that $(\Delta f)^T$ is zero. We may suppose by contradiction $(\Delta f)^T\neq 0 $ at some point $p$, this implies it is nonzero in a small neighborhood $U$ of $p$.\\
Then we choose a smooth function $\psi \geq 0$ of compact support contained in $U$, such that $\psi(p)>0$, and define the deformation
\[ \xi(x,y,t)=\exp_{f(x,y)}\bigl(t\psi(x,y)(\Delta f)^T(x,y)\bigr)\]
By substituting $\xi$ in the expression \eqref{eq:tglaplnull1} we obtain 
\begin{align*}
	\left.\frac{dE(f_t)}{dt}\right|_{t=0}  &=  -2\int_{\supp \psi } \psi\, |(\Delta f)^T |^2dx\wedge dy
		<0 
\ \ , 
\end{align*}
where we have the strict inequality since we have both $\psi>0$ and $(\Delta f)^T \neq 0$ on $\supp \psi \subset U$. This is absurd because we showed that the leftmost side of the last expression is exactly zero.

\end{proof}
Let us fix now a conformal parametrization of our surface,
\[\sigma : \mathbb{D}\to \M,\ \] 
and let 
\[N:\mathbb{D}\to \M\] 
be the gauss map of the surface.
We want to prove the following theorem:
\begin{thm}\label{th:nablh}
	\[(\Delta N)^T = 2(\nabla H)\ ,\] 
	where $H$ is the mean curvature of the surface.
\end{thm}
In order to prove it, we need a lemma.
\begin{lemma}\label{lm:trnabl}
	Let $B$ a 1-1 tensor on a Riemannian surface $M$, $v\in T_pM$, and denote with $\nabla$ the Levi-Civita connection of $M$. Then
	\[\tr(\nabla_v B)=d(\tr B)(v)\ .\] 
\end{lemma}
\begin{proof}
	Let us suppose  the dimension of the manifold to be $2$ for simplicity. Let $(e_i)_i$ be an orthonormal frame around $p$.\\
	Let us observe
	\begin{align}\label{eq:trb1}
	d\tr(B)(v)
	&= v\left(\sum_i \langle B(e_i),e_i\rangle\right)\notag\\
	&= \sum_i \left\langle \nabla_v(B(e_i)),e_i\right\rangle
	   + \sum_i \left\langle B(e_i),\nabla_v e_i\right\rangle\notag\\
	&= \sum_i \left\langle (\nabla_vB)(e_i),e_i\right\rangle
	   + \sum_i \left[\left\langle B(\nabla_v e_i),e_i\right\rangle
   +  \left\langle B(e_i),\nabla_v e_i\right\rangle\right]\notag\\
	&= \tr(\nabla_v B )
	   + \sum_i \left[\left\langle B(\nabla_v e_i),e_i\right\rangle
	   +  \left\langle B(e_i),\nabla_v e_i\right\rangle\right]
	\ \ .
	\end{align}
	So that we have our thesis if we show that the second term of the expression in \eqref{eq:trb1} is zero.
	Suppose  
	\begin{align*}
		\nabla_v e_1 = ae_1 + be_2,\qquad \nabla_v e_2 = ce_1+de_2\ .
	\ \ , 
	\end{align*}
	for certain coefficients $a,b,c,d\in \R$.\\
	Observe then that 
	\begin{align*}
	\left<e_i,e_j\right> &= \delta_{ij}\\
	\then\quad v\left(\left<e_i,e_j\right>\right) &= v(\delta_{ij}) = 0\\
	\then\quad \left<\nabla_v e_i,e_j\right> + \left<e_i,\nabla_v e_j\right> &= 0\\
	\then\quad \left<\nabla_v e_i,e_j\right> &= -\left<e_i,\nabla_v e_j\right>.
	\end{align*}
	By setting $i=j$ we obtain  $\left<\nabla_v e_1,e_1 \right> =a= \left<\nabla_v e_2,e_2 \right> = d = 0$ and if $i\neq j$ we get $\left<\nabla_v e_1,e_2 \right> = b = -\left<\nabla_v e_2 ,e_1\right> = -c$ . Hence
	\[\nabla_v e_1 = be_2,\qquad \nabla_v e_2 = -be_1.\]
	Now we can rewrite the second term of  \eqref{eq:trb1},
	\begin{align*}
	   \left\langle B(\nabla_v e_1),e_1\right\rangle
	   +  \left\langle B(e_1),\nabla_v e_1\right\rangle
	   + \left\langle B(\nabla_v e_2),e_2\right\rangle
	   +  \left\langle B(e_2),\nabla_v e_2\right\rangle = \\
	   b\left\langle B(e_2),e_1\right\rangle
	   +  b\left\langle B(e_1),e_2\right\rangle
	   - b\left\langle B(e_1),e_2\right\rangle
	   - b\left\langle B(e_2),e_1\right\rangle = 0
	\ \ . 
	\end{align*}
	Which proves the lemma.
\end{proof}

We are ready to prove theorem \ref{th:nablh}.
\begin{proof}[Proof of theorem \ref{th:nablh}]
	First we recall that if $A$ is the shape operator, then $N_x = A\partial_x$ and $N_y= A\partial_y$. If we assume that $\I = e^{2\eta}(dx^2+dy^2)$ then $e^{-\eta}\partial_x$, $e^{-\eta}\partial_y$ is an orthonormal frame.\\
	By applying the covariant derivatives along $\partial_x$ and $\partial_y$ to $N_x$ and $N_y$, respectively, we obtain
	\begin{align*}
		N_{xx}^T &= (\nabla_{\partial_x} A) (\partial_x) + A(\nabla_{\partial_x}\partial_x)\\
		N_{yy}^T &= (\nabla_{\partial_y}A) (\partial_y) + A(\nabla_{\partial_y}\partial_y)
	\ \ . 
	\end{align*}
	So we obtain
	\begin{align}\label{eq:nablh1}
	(\Delta_\I N)^T
	&=
	e^{-2\eta}\bigl((\nabla_{\partial_x}A)(\partial_x)+(\nabla_{\partial_y}A)(\partial_y)\bigr) \notag\\
	&\qquad + e^{-2\eta}A\bigl(\nabla_{\partial_x}\partial_x+\nabla_{\partial_y}\partial_y\bigr)
	\ \ .
	\end{align}
	Now let us show that $\nabla_{\partial_x}\partial_x + \nabla_{\partial_y}\partial_y = 0$.\\
	First, observe that since $\partial_x,\partial_y$ are orthogonal, $\left<\partial_x,\partial_y \right> = 0$ we have $\left<\nabla_{\partial_y}\partial_x,\partial_y \right> = - \left<\partial_x, \nabla_{\partial_y}\partial_y \right>$. Furthermore, from the expression of the metric we have $\left<\partial_x, \partial_x \right> = e^{2\eta} = \left<\partial_y,\partial_y \right>$ so that $\partial_y(\left<\partial_x,\partial_x \right>) = \partial_x(\left<\partial_y,\partial_y \right>)$ and also, the connection is symmetric so that $\nabla_{\partial_x} \partial_y = \nabla_{\partial_y} \partial_x$. Then,	
	\[
	\begin{aligned}[t]
	\left< \nabla_{\partial_x} \partial_x, \partial_y\right>
	&= -\left<\partial_x, \nabla_{\partial_x}\partial_y \right>\\
	&= -\frac{1}{2}\partial_y\left<\partial_x,\partial_x \right> \\
	&= -\frac{1}{2}\partial_y\left<\partial_y,\partial_y \right> \\
	&= -\left<\nabla_{\partial_y}\partial_y, \partial_y \right>
	\ \ ,
	\end{aligned}
	\qquad
	\begin{aligned}[t]
	\text{}\left<\nabla_{\partial_x}\partial_x,\partial_x \right>
	&= \frac{1}{2}\partial_x\left<\partial_x,\partial_x \right> \\
	&= \frac{1}{2}\partial_x\left<\partial_y,\partial_y \right> \\
	&= \left<\nabla_{\partial_x}\partial_y,\partial_y \right> \\
	&= \left<\nabla_{\partial_y}\partial_x,\partial_y \right> \\
	&= -\left<\partial_x,\nabla_{\partial_y}\partial_y \right>
	\ \ .
	\end{aligned}
	\]
	At this point, by developing on the base $\partial_x,\partial_y$ it is clear that
	 \[\nabla_{\partial_x}\partial_x+\nabla_{\partial_y}\partial_y=0\ .\]
	So in \eqref{eq:nablh1} we obtain
	\begin{align}\label{eq:nablh2}
		(\Delta N)^T
		=
		e^{-2\eta}\bigl((\nabla_{\partial_x}A)(\partial_x)+(\nabla_{\partial_y}A)(\partial_y)\bigr)
	\ . 
	\end{align}
	Since the shape operator $A$ is defined by $\II(Y,Z) = \left<AY,Z \right>$, we can relate the covariant derivatives of $A$ and $\II$ as follows:
	\begin{align*}
		(\nabla_X\II) (Y,Z)  &= X(\II(Y,Z)) -\II(\nabla_XY,Z)-\II(Y,\nabla_X Z) \\
		&= X(\left<AY,Z \right>)-\left<A(\nabla_X Y),Z \right>-\left<AY,\nabla_XZ \right> \\
		&= \left<\nabla_X(AY),Z \right> + \cancel{\left<AY,\nabla_X Z \right>}-
		\left<A(\nabla_XY),Z \right> -\cancel{\left<AY, \nabla_XZ \right>}\\
		&= \left<(\nabla_XA)(Y),Z \right>
	\ \ . 
	\end{align*}
	Then we can apply the Codazzi equation in the following form:
	\begin{align}\label{eq:nablh3}
		\left<(\nabla_{\partial_y}A)(\partial_y),\partial_x \right> &=  
		(\nabla_{\partial_y}\II) (\partial_y,\partial_x) \notag\\
		&= (\nabla_{\partial_y}\II) (\partial_x,\partial_y) \notag\\
		&= \left< (\nabla_{\partial_x}A)(\partial_y),\partial_y  \right>
	\ . 
	\end{align}
	So, finally, by taking the scalar product of \eqref{eq:nablh2} with $\partial_x$, we obtain
	\begin{align*}
		\left< (\Delta N)^T, \partial_x \right>
		&= e^{-2\eta} \left[ \left< (\nabla_{\partial_x}A) (\partial_x) ,\partial_x \right> + \left< (\nabla_{\partial_y}A) (\partial_y) ,\partial_x \right> \right] \\
		\commb{\qref{eq:nablh3}}\qquad
		&= e^{-2\eta} \left[ \left< (\nabla_{\partial_x}A) (\partial_x) ,\partial_x \right> + \left< (\nabla_{\partial_x}A) (\partial_y) , \partial_y \right> \right]  \\
		&= \left< (\nabla_{\partial_x}A) (e^{-\eta }\partial_x) , e^{-\eta} \partial_x \right> +
		\left< (\nabla_{\partial_x}A) (e^{-\eta} \partial_y) , e^{-\eta}\partial_y \right>\\
		&= \tr ( \nabla_{\partial_x}A )  \\
		\commb{Lemma \ref{lm:trnabl}}\quad &= d\tr ( A ) (\partial_x)  \\
		&= 2 \partial_x(H) = 2H_x
	\ .	
	\end{align*}
	Analogously we can show that 
	\begin{align*}
		\left< (\Delta N) ^T, \partial_y\right> = 2 H_y	
	\ \ . 
	\end{align*}
	Therefore we have the thesis.
\end{proof}
We have now all the pieces needed to prove the central theorem of this subsection
\begin{thm}\label{th:gaussmapharmiffconsth}
	Let $\sigma:(\mathbb{D},e^{2\eta}(dx^2+dy^2)) \to  S \subset  M$ a parametrization of a space-like surface and denote with  $N:\mathbb{D}\to \M$ the corresponding Gauss map. Then $S$ has constant mean curvature if and only if $N$ is harmonic.
\end{thm}
\begin{proof}
	Just observe that for theorem  \ref{th:nablh} we have 
	\[(\Delta N)^T = 2 \nabla H. \]
	Then recall from corollary \ref{cor:tglapnull} that $N$ if and only if the LHS in the last equation is zero, which in turn is equivalent to $H$ being constant. 
\end{proof}
\subsection{Harmonic maps between Riemann surfaces}
We can formulate the notion of harmonicity in a slightly more general setting. Let $(\Sigma_1,g)$ and $(\Sigma_2,h)$ be two Riemann surfaces, and let $z$ and $w$ be local conformal coordinates on $\Sigma_1$ and $\Sigma_2$, respectively, so that the metrics are written as
\begin{align*}
	g=e^{2\eta}|dz|^2 = \lambda^2|dz|^2,
	\qquad
	h=\rho^2(w)|dw|^2
	\ \ .
\end{align*}
Let $f:\Sigma_1\to \Sigma_2$ be a smooth map, and denote for simplicity by $w$ its local expression in the chosen coordinates. In the last section we set $\Sigma_1 = \mathbb{D}$ and $\Sigma_2=\mathbb{H}$ and defined the energy density of $f$ as
\begin{align*}
	e(f)=\frac{1}{\lambda^2}\bigl(f_x\cdot f_x+f_y\cdot f_y\bigr)
	\ \ .
\end{align*}
Since the metric on the target is $\rho^2(w)|dw|^2$, we have
\begin{align*}
	f_x\cdot f_x=\rho^2(w)|w_x|^2,
	\qquad
	f_y\cdot f_y=\rho^2(w)|w_y|^2,
\end{align*}
and therefore
\begin{align*}
	e(f)=\frac{\rho^2(w)}{\lambda^2}\bigl(|w_x|^2+|w_y|^2\bigr)
	\ \ .
\end{align*}
Now, if we introduce the complex coordinate $z=x+iy$, as before, we have $w_z=\frac12(w_x-iw_y)$ and $w_{\bar z}=\frac12(w_x+iw_y)$. Thus
\begin{align*}
	w_x=w_z+w_{\bar z},
	\qquad
	w_y=i(w_z-w_{\bar z})
	\ \ .
\end{align*}
Hence
\begin{align*}
	|w_x|^2+|w_y|^2
	&=
	|w_z+w_{\bar z}|^2+|i(w_z-w_{\bar z})|^2\\
	&=
	|w_z+w_{\bar z}|^2+|w_z-w_{\bar z}|^2\\
	&=
	2\bigl(|w_z|^2+|w_{\bar z}|^2\bigr),
\end{align*}
so that the energy density can be written as
\begin{align*}
	e(f)=2 \frac{\rho^2(w)}{\lambda^2}\bigl(|w_z|^2+|w_{\bar z}|^2\bigr)
	\ \ .
\end{align*}
Then the energy of the map, when defined, becomes
\begin{align*}
	E(f)
	&:= \int_{\Sigma_1}\lambda^2(z)e(f(z))\, i\, dzd\overline{z}\\
	&= \int_{\Sigma_1}\rho^2(w)\bigl(|w_z|^2+|w_{\bar z}|^2\bigr)\, i\, dzd\bar z
	\ \ .
\end{align*}
Like we already did before, we can define a harmonic map as a critical point for any admissible deformation (that is, a perturbation with compact support). The correspective Euler-Lagrange equation of such a problem is 
\[ w_{z\bar z}+2\frac{\rho_w}{\rho}w_zw_{\bar z}=0\ .\]
For a complete derivation we redirect the reader to \cite{jost2006compact}.
An important observation is that the local harmonic map equation does not involve the conformal factor $\lambda$ of the metric on the domain. This reflects the fact that, in dimension two, harmonicity depends only on the conformal class of the metric on the source surface.
\subsection{Energy densities and the Hopf differential}
\paragraph{Energy densities and Jacobian}\mbox{}\\
In addition to the energy density of the map $f$, it is convenient to introduce two more refined quantities, the holomorphic and anti-holomorphic energy densities. These objects will let us also express the Jacobian of $f$ in a convenient way.
\begin{defn}
	We define the holomorphic and antiholomorphic energy density of a smooth map $f:\Sigma_1\to \Sigma_2$ respectively as	
\begin{align*}
	\mathcal H(f):=\frac{\rho^2(f)}{\lambda^2}|f_z|^2,
	\qquad\text{and}\qquad
	\mathcal L(f):=\frac{\rho^2(f)}{\lambda^2}|f_{\bar z}|^2
\ \ .
\end{align*}
\end{defn}
It is immediate to note that 
\[e(f) = \mathcal H(f)+\mathcal L(f)\ .\] 
The Jacobian of the map is by definition
\begin{align*}
	J(f) =  \frac{\rho^2(f)}{\lambda^2}(|f_z|^2-|f_{\overline{z}}|^2)
\ \ . 
\end{align*}
So that we also have 
\begin{align}\label{eq:defjacob}
	J(f) = \mathcal H(f) - \mathcal L(f)\ .
\ \ . 
\end{align}
\paragraph{Quadratic differentials}\mbox{}\\
To each map between Riemann surfaces we can associate a quantity that carries information about how far the map is from being conformal.
\begin{defn}
	Given $\Sigma$ a Riemann surface we say that a symmetric 0-2 tensor field on $\Sigma$ with values in $\C$ is compatible with the complex structure if it can be expressed locally in isothermal coordinates as $\Phi = \varphi(z)dz^2$. We say that $\Phi$ is holomorphic if $\varphi$ is a holomorphic function.
\end{defn}
It is clear that if we ha a change of coordiante $z \mapsto w = \gamma(z)$, then we can express the differentials as $dw = \gamma'(z)dz$. This means that the tensor product $dz^2$ will transform as 
\[ dw^2 = \gamma'(z)^2 dz^2 .\]
As it is usual the case, a set of local representations of holomoprhic quadratic differentials $(dw_\alpha^2)_\alpha$ determines a holomorphic quadratic differential on the whole surface if they satisfy the transformation rule above.\\

To each map between Riemann surfaces, we can associate a quadratic differential, which gives us in some sense a characterization of harmonic local diffeomorphisms
\begin{lemma}\label{lm:exholdiff}
	Let $f:\Sigma_1\to \Sigma_2$ a harmonic map between Riemann surfaces, with $\Sigma_1$ being endowed with a metric of nonpositive curvature. Let $z$ be a conformal parameter on $\Sigma_1$. Then if $w$ is a local representation of $f$,
	\[q = \rho^2(w) w_zw_{\overline{z}} dz^2\] is a holomorphic quadratic differential on $\Sigma_1$.
\end{lemma}
\begin{proof}
	Let us show first that the expression of $q$ doesn't change under conformal transformations of the domain and the target.\\

	First let us consider a conformal transformation on the domain, $z=\psi(z')$. Then $dz=\psi'(z') dz'$, $dz^2 =(\psi(z'))^2 () dz')^2$, 
	\[w_{z'}=w_z \psi'(z'),\qquad\text{and}\qquad \overline{w}_{z'}=\overline{w}_z\psi'(z')\ .\] 
	So the metric transforms as
	 \begin{align*}
		\rho^2(w)w_{z'}\overline{w}_{z'}  &= \rho^2(w)w_{z}\overline{w}_{z}(\psi'(z'))^2 (dz') ^2 \\
		&= \rho^2(w)w_z \overline{w}_z dz^2
	\ . 
	\end{align*}
	Hence $q$ changes as 
	\begin{align*}
		q = \rho^2(w)w_z\overline{w}_z dz^2 &= \rho^2(w) w_z \overline{w}_z(\psi'(z'))^2 (dz') ^2 \\
		&= \rho^2(w) w_z \overline{w}_z dz^2 
	\ . 
	\end{align*}

	Then we consider a conformal transformation of the target, $w = \phi(w')$. As before we have $dw = \phi'(w') dw'$, and
	\begin{align*}
		\rho^2(w)|dw|^2 &= \rho^2(\phi(w'))|\phi'(w')|^2|dw'|^2 \\
		&= \rho^2(w')|dw'|^2 
	\ . 
	\end{align*}
	Then
	\begin{align*}
		q &= \rho^2(w)w_z \overline{w}_z dz^2  \\
		  &= \rho^2(\phi(w'))(\phi'(w')w'_z \overline{\phi'(w')}\overline{w}_z) dz^2 \\
		  &= \rho^2(\phi(w'))(|\phi'(w)|^2w_z\overline{w}_zdz^2 \\
		  &= \rho^2(w')w_z\overline{w}_zdz^2
	\ . 
	\end{align*}

	Finally, if $f$ is harmonic, we may show that $\rho^2(w) w_z\overline{w}_z $ is holomorphic
	\begin{align*}
		\frac{\partial  }{\partial \overline{z}} (\rho^2(w)w_z\overline{w}_z) &=
		\frac{\partial  }{\partial \overline{z}} (\rho^2(w)) w_z w_{\overline{z}} +\rho^2(w) w_{z\overline{z}} \overline{w}_z + \rho^2(w)w_z \overline{w}_{z\overline{z}}\\
		&= 2\rho(w)(\rho_w(w)w_{\overline{z}} +\rho_{\overline{w}}(w)\overline{w}_{\overline{z}})w_z\overline{w}_z+ \rho^2(w) w_{z\overline{z}} \overline{w}_z + \rho^2(w)w_z \overline{w}_{z\overline{z}}\\
		&= \overline{w}_z(\rho^2(w)w_{z\overline{z}}+2\rho(w) \rho_w(w)w_{\overline{z}}w_z)+\\
		&\qquad+ \overline{w}_z(\rho^2(\overline{w})\overline{w}_{z\overline{z}}+2\rho(\overline{w}) \rho_{\overline{w}}(\overline{w})\overline{w}_{\overline{z}}\overline{w}_z)\\
		&= \overline{w}_z(\rho^2(w)w_{z\overline{z}}+2\rho(w) \rho_w(w)w_{\overline{z}}w_z)+\\
		&\qquad+ \overline{w}_z\overline{(\rho^2(w)w_{z\overline{z}}+2\rho(w) \rho_w(w)w_{\overline{z}}w_z)} = 0\\
	\ , 
	\end{align*}
	where the last equality holds by applying the local equation for harmonicity.
\end{proof}
We can actually give a decomposition of any symmetric 0-2 tensor field
\begin{lemma}\label{lm:tfdec}
	Let $\omega$ a symmetric 0-2 tensor field on $(\Sigma,g)$ with values in  $\C$ . Then $\omega$ can be decomposed uniquely as
	\[\omega = \omega^{(2,0)}+\omega^{(1,1)}+\omega^{(0,2)},\] 
where $\omega^{(2,0)}$ and $\overline{\omega^{(0,2)}}$ are quadratic differentials compatible with the complex structure and $\omega^{(1,1)}$ is a multiple of the metric $g$. 
\end{lemma}
\begin{proof}
	Just observe that once fixed local coordinates $u,v$ the symmetric forms $dz^2 = du^2-dv^2+2idudv$, $d\bar z = du^2-dv^2-2idudv$ and $|dz|^2 = du^2+dv^2$ are three linearly independent elements in a three-dimensional space, hence a base. 
\end{proof}
We can use this lemma to find a decomposition of the pull-back metric $(f^*(h))$. What we find is actually the quadratic differential we defined in lemma \ref{lm:exholdiff}
\begin{lemma}
	Let $f:(\Sigma_1,g)\to (\Sigma_2,h) $ a map two Riemann surfaces with $\Sigma_1$ being hyperbolic. Let $q$ be the quadratic differential associated to $f$ as in lemma \ref{lm:exholdiff}. Then 
	\[q = (f^*(h))^{(2,0)}\ .\] 
	Moreover we have that
	\[ f^*(h) = q + eg +\bar q\ .\] 
\end{lemma}
\begin{proof}
	Let us assume as usual that $g=\lambda^2|dz|^2$ and $h=\rho^2|dw|^2$ are the metrics respectively on $\Sigma_1$ and $\Sigma_2$, and that $w=w(z)$ is a local representation of the map $f$. Then we have
	\begin{align*}
		f^*(dw) = dw = w_z dz + w_{\overline{z}}d\overline{z}\qquad\text{and}\qquad f^*(d\overline{w})=\overline{w}_zdz+\overline{w}_{\overline{z}}d\overline{z}
	\ . 
	\end{align*}
	Now, if we pull back $h$ we obtain
	\begin{align*}
		f^*(h) &= \rho^2(w)|f^*(dw)|^2  \\
		&=  \rho^2(w)(w_zdz+w_{\overline{z}}d\overline{z})(\overline{w}_zdz+\overline{w}_{\overline{z}}d\overline{z}) \\
		&= \rho^2(w)\big(w_z\overline{w}_zdz^2 +(|w_z|^2+|w_{\overline{z}}|^2) |dz|^2+ \overline{w}_{\overline{z}}w_{\overline{z}}d\overline{z}^2\big)
	\ . 
	\end{align*}
	Hence proving the decomposition.
\end{proof}
This motivates the definition of the Hopf differential
\begin{defn}
	Given $f:(\sigma_1,g) \to  (\Sigma_2,h) $ between two Riemann surfaces, we define the Hopf differential as the quadratic differential
	\[\Hopf(f) = (f^*(h))^{(2,0)}\ .\] 
\end{defn}
This lets us state an important result, that is a partial inverse of lemma \ref{lm:exholdiff}.
\begin{cor}\label{cor:diffeopresthenharm}
	If $f$ is a local diffeomorphism, so that it is orientation-preserving, then if $\Hopf(f)$ is holomorphic, $f$ is harmonic.
\end{cor}
\begin{proof}
	In the proof of lemma \ref{lm:exholdiff} we showed that
	\begin{align*}
		\frac{\partial}{\partial \overline{z}} (\rho^2(w)w_z\overline{w}_z)
		&=
		\overline{w}_z\bigl(\rho^2(w)w_{z\overline{z}}+2\rho(w)\rho_w(w)w_{\overline{z}}w_z\bigr)\\
		&\qquad+
		w_z\overline{\bigl(\rho^2(w)w_{z\overline{z}}+2\rho(w)\rho_w(w)w_{\overline{z}}w_z\bigr)}.
	\end{align*}
	Since $\Hopf(f)$ is holomorphic, the LHS is zero. Then set
	\[
	F:=\rho^2(w)w_{z\overline{z}}+2\rho(w)\rho_w(w)w_{\overline{z}}w_z.
	\]
\begin{align}
	0
	&=
	\frac{\partial}{\partial \overline z}\bigl(\rho^2(w)w_z\overline{w_z}\bigr)\notag\\
	&\Rightarrow
	\overline{w_{\overline z}}F+w_z\overline F=0\notag\\
	&\Rightarrow
	\overline{w_{\overline z}}F=-w_z\overline F\label{eq:hopfhol1}\\
	&\Rightarrow
	w_{\overline z}\overline{F}=-\overline{w_z}F\label{eq:hopfhol2}\ .
\end{align}
By multiplying \eqref{eq:hopfhol1} and \eqref{eq:hopfhol2} we obtain
\begin{align*}
	|w_{\overline z}|^2|F|^2=|w_z|^2|F|^2,
\end{align*}
hence
\begin{align*}
	\bigl(|w_z|^2-|w_{\overline z}|^2\bigr)|F|^2=0.
\end{align*}
	Since $f$ is orientation-preserving local diffeomorphism, remembering that $\overline{w_z} = \overline{w}_{\overline{z}}$,
	\begin{align*}
		|w_z|^2-|w_{\overline{z}}|^2=J(f)>0,
	\end{align*}
	hence $F=0$ and $f$ is harmonic. 
\end{proof}
We can calculate, pointwise, a norm for a quadratic differential. Fix $(e_i)_i$ a local orthonormal frame on $(\Sigma_1,g)$. Let us define the $g$ norm of the quadratic differential $q $ as
\[ \|q\|_g = \frac{1}{4}\sum_{i,j} |q(e_i,e_j)|^2\ . \] 
It is possible to rewrite this quantity in terms of the energies of the map $f$. If the metric is $\lambda^2|dz|^2$, two orthonromal vectors are $e_1= \frac{1}{\lambda} \frac{\partial  }{\partial x} $ and $e_2 = \frac{1}{\lambda}\frac{\partial  }{\partial y} $, hence
\[dz^2(e_1,e_1)= \frac{1}{\lambda^2},\ dz^2(e_1,e_2)=i \frac{1}{\lambda^2},\ dz^2(e_2,e_2)=-\frac{1}{\lambda^2}\ .\] 
So that we have
\begin{align*}
	\|q\|_{g}^2 &= \frac{1}{4}\sum_{ij} |q(e_i,e_j)|^2 \\
	&= \frac{1}{4}\left( |q(e_1,e_1)|^2+2|q(e_1,e_2)|^2+|q(e_2,e_2)|^2 \right)  \\
	&= \frac{1}{4}|q(z)|^2\left(|dz(e_1,e_1)|^2 +2|q(e_1,e_2)|^2+|q(e_2,e_2)|^2\right)\\
	&=  \frac{|q(z)|^2}{\lambda^4(z)}
\ \ . 
\end{align*}

If we apply calculate the norm of the Hopf differential of a smooth $f:\Sigma_1\to \Sigma_2$, with local representation $w=w(z)$, 
\[\|\Hopf(f)\|_g^2(z) = \frac{\rho^4(w)}{\lambda^4} |w_z|^2|w_{\overline{z}}|^2\ .\] 
\paragraph{The second fundamental form and the Hopf differential of the Gauss map}\mbox{}\\
We now present some observations that will illustrate how the Hopf differential of the Gauss map is related to the second fundamental form.
\begin{lemma}\label{lm:lemmaqforms}
	Let $x,y$ be some coordinates. Then
	\[
		a(dx^2-dy^2) +2bdxdy = \frac{1}{2}[(a-ib)(dx+idy)^2+(a+ib)(dx-idy)^2]
	\]
\end{lemma}
\begin{proof}
	By calculation, (remember the shorthand $dxdy = \frac{1}{2} (dx \otimes dy+dy\otimes dx),\ dx^2 = dx\otimes dx,\ dy^2=dy\otimes dy$) 
	\begin{align*}
		(a-ib)(dx+idy)^2&= (a-ib) (dx^2-dy^2+2idxdy)\\
		(a+ib)(dx-idy)^2&=(a+ib)(dx^2-dy^2-2idxdy)
	\ , 
	\end{align*}
	so that 
	\begin{align*}
		(a-ib)&(dx+idy)^2+(a+ib)(dx-idy)^2= \\
		&= adx^2-ady^2+2aidxdy-ibdx^2+ibdy^2+2bdxdy\\
		&+adx^2-ady^2-2aidxdy+ibdx^2-ibdy^2+2bdxdy=\\
		&= 2a(dx^2-dy^2)+4bdxdy\ .
	\end{align*}
\end{proof}
Hence if we set $z = x+iy, \bar z = x-iy$ we obtain
\[
	a(dx^2-dy^2)+2bdxdy = \frac{1}{2}\big[(a-ib)dz^2+(a+ib)d\bar z^2\big]
\ . 
\]

Let us now take $\alpha$ a symmetric 2-form on  $S$. Then there is a morphism of the tangent bundle $L:TM\to TM$ such that 
\[
	\alpha(\cdot ,\cdot ) = \I(L\cdot ,\cdot )
\ . 
\]
Which is self-adjoint, since $\alpha$ is symmetric by hypothesis. 
$L$ is at every point a linear endomorphism, and we can decompose it using $\I$ in a traceless part and a diagonal part,
\[
	L = \frac{\tr L}{2}\Id +L_0
\ . 
\]
Now, let us suppose that $(x,y)$ are isothermal coordinates. We can write $L_0$ as
\begin{align*}
	L_0(\partial_x) &= a \partial_x +b \partial_y\\
	L_0(\partial_y) &= c\partial_x +d\partial_y
\ , 
\end{align*}
and since $L_0$ is symmetric and traceless, we have $d = -a$ and $c = b$. Hence we can represent $L_0$ with the matrix
\begin{align*}
	[L_0] = 
	\begin{bmatrix} a&b \\ b&-a \end{bmatrix} 
\ . 
\end{align*}
Now, as in lemma \ref{lm:tfdec} we may suppose to write $\alpha$ as 
\begin{align}\label{eq:dec}
	\alpha = \alpha^{(2,0)} + \alpha^{(1,1)} + \alpha^{(0,2)}
\ . 
\end{align}
We may observe that, in the decomposition \eqref{eq:dec}, from the theorem we have that $\alpha^{(2,0)}$ and $\alpha^{(0,2)}$ are locally multiples of  $dz^2$ and $d\bar z^2$ respectively, which are traceless. Hence we observe
\begin{align*}
	\alpha = \alpha^{(2,0)} + \alpha^{(1,1)} + \alpha^{(0,2)} &= \I(L\cdot ,\cdot ) \\
		&= \I\left(\left( \tfrac{1}{2}{\tr L}\Id + L_0\right)\cdot ,\cdot \right) \\
		&= \tfrac{1}{2}\tr(L) \I(\cdot ,\cdot ) +\I(L_0 \cdot ,\cdot )
\ . 
\end{align*}
By separating the traceless part and the trace part we have
\begin{align}\label{eq:trthm3}
	\alpha^{(1,1)} = \tfrac{1}{2}\tr(L)\I, \qquad \alpha^{(2,0)}+\alpha^{(0,2)} = \I(L_0\cdot ,\cdot )
\ . 
\end{align}
Also, again from \ref{lm:tfdec}, since $\alpha$ is real valued, we have $\alpha^{(0,2)}=\overline{\alpha^{(2,0)}}$.
We may now prove the following important lemma
\begin{lemma}\label{lm:hopfdifflocalform}
	With the notation above, it holds that
\begin{align*}
	\alpha^{(2,0)} = \tfrac{1}{2}(a-ib) dz^2
\ . 
\end{align*} 
\end{lemma}
\begin{proof}
	
Let us fix $v = v_x \partial_x +v_y \partial_y$, by looking at the associated quadratic form to $\I(L_0\cdot ,\cdot )$,
 \begin{align*}
	 \I(L_0(v_x\partial_x+v_y\partial_y),v_x\partial x+ v_y \partial_y)
	&= \I(v_xL_0\partial_x+v_yL_0\partial_y,v_x\partial x+ v_y \partial_y)\\
	&= \I(v_x(a\partial_x+b\partial_y)+v_y(b\partial_x-a\partial_y),v_x\partial x+ v_y \partial_y)\\
	&= e^{2u} \left(   a v_x^2+bv_xv_y+bv_xv_y-av_y^2\right)
 \ , 
 \end{align*}
 we have
\begin{align*}
	\I(L_0\cdot ,\cdot ) &= a(dx^2-dy^2) +2bdxdy\\
	\commb{Lemma \ref{lm:lemmaqforms}}\hspace{3mm}
	&=\tfrac{1}{2}(a-ib)(dx+idy)^2 +\tfrac{1}{2}(a+ib)(dx-idy)^2
\ . 
\end{align*}
Therefore 
\begin{align}\label{eq:trthm4}
	\alpha^{(2,0)} = \tfrac{1}{2}(a-ib)dz^2
\ , 
\end{align}
as we intended.
\end{proof}

Let now $\alpha$ be the second fundamental form $\II$, so that $L$ is the shape operator $A$. We consider the Gauss map $N:\mathbb{D}\to \mathbb{H}$ and the pull-back of the metric of $\mathbb{H}$ through $N$, which we can write as
\begin{align}\label{eq:trthm2}
	N^*(g_{\mathbb{H}^2})(v,w) &= \left<D_vN,D_wN \right> \notag\\
	\commb{def. of $ A$}\hspace{2cm}&=  \I(A(v),A(w)) \notag\\
	\commb{$A$ is self-adjoint}\hspace{2cm}&= \I(A^2v,w) 
\ . 
\end{align}
We can apply Cayley-Hamilton's theorem on $p_A(x) = x^2-\tr(A)x+\det(A)$, obtaining
\begin{align*}
	p_A(A) = 0\Rightarrow A^2 &= \tr(A) A - \det(A) \1
\ . 
\end{align*}
Recalling that $\tr(A)=2H$ and $\det(A)=-K$, we obtain
\begin{align}\label{eq:trthm1}
	A^2 &= 2HA + K\Id \notag\\
	&= 2H(A_0+H\Id)+K\Id \notag\\
	&= 2HA_0 +(2H^2+K)\Id
\ . 
\end{align}
Which implies in particular that $e = 2H^2+K $.
Now, the relations \eqref{eq:trthm2} and \eqref{eq:trthm1} along with the general \eqref{eq:trthm3} give us
\[
	N^*(g_\mathbb{H})^{(2,0)} + \overline{N^*(g_\mathbb{H})^{(2,0)}} = 2H\,\I(A_0\cdot,\cdot)
\ . 
\]
So that by applying the identity \eqref{eq:trthm4} to $\alpha=2H\,\II_0$ we obtain
\begin{align*}
	N^*(g_\mathbb{H})^{(2,0)} = H(a-ib)dz^2
\ . 
\end{align*}
And we can rewrite
\begin{align*}
	\Hopf(N)+\overline{\Hopf(N)} = 2H\,\I(A_0\cdot ,\cdot )
\ . 
\end{align*}
As usual let us write $\II$ locally as
\begin{align*}
	\II =  h_{ij}dx^idx^j,
	\quad\text{and suppose}\quad
	\Hopf(N)=\varphi\,dz^2.
\end{align*}
Since $H=1$ and $\I=e^{2w}(dx^2+dy^2)$, we have
\begin{align*}
	h_{11}+h_{22}=2e^{2w}.
\end{align*}
Moreover, from
\begin{align*}
	2\II_0=\Hopf(N)+\overline{\Hopf(N)}
	=\varphi\,dz^2+\overline{\varphi}\,d\bar z^2
\end{align*}
and Lemma \ref{lm:lemmaqforms}, writing $\varphi=a-ib$, we obtain
\begin{align*}
	2\II_0
	&=2a(dx^2-dy^2)+4bdxdy\\
	&=(h_{11}-h_{22})(dx^2-dy^2)+4h_{12}dxdy,
\end{align*}
hence
\begin{align*}
	h_{11}-h_{22}=2\Re(\varphi),
	\qquad
	h_{12}=-\Im(\varphi).
\end{align*}
We note that these relations can be used to reconstruct the differential $\varphi dz^2$, by solving the linear
\begin{align}\label{eq:gausspde1}
	\begin{cases}
		h_{11}+h_{22} = 2e^{2w}\\
		h_{11}-h_{22} =  2\Re(\varphi)\\
		h_{12}   = -\Im(\varphi)
	\end{cases}
\ \ .
\end{align}
\paragraph{The Codazzi equation is equivalent to $\Phi$ holomorphic}\mbox{}\\
The relations obtained above connect the second fundamental form with the Hopf differential of the Gauss map. In particular, when the mean curvature is constant, they allow us to prove that the Codazzi equation for the second fundamental form is equivalent to the holomorphicity of the Hopf differential of the Gauss map.\\

First we need to remind that from the compatibility of $\nabla$ with the metric we get $(\nabla_X\I)(Y,Z) = 0$, so that
\begin{align*}
	(\nabla_X\II)(Y,Z) &= (\nabla_X(H\I +\II_0))(Y,Z) \\
	\commb{$H$ is constant, $\nabla \I = 0$}\hspace*{1cm} &= (\nabla_X\II_0)(Y,Z)
	\ . \hspace*{2cm}
\end{align*}
Hence the Codazzi equation can be rewritten exclusively in terms of the traceless part
\begin{align*}
	(\nabla_X\II)(Y,Z) = (\nabla_Z \II) (X,Y)\quad \iff \quad (\nabla_X\II_0)(Y,Z) = (\nabla_Z \II_0) (X,Y) 
\ . 
\end{align*}

Let us now choose local isothermal coordinates $(x,y)$. Since $A_0$ is self-adjoint and traceless, with respect to the frame $(\partial_x,\partial_y)$ it is represented by a matrix of the form
\begin{align*}
	[A_0]=
	\begin{pmatrix}
		a & b\\
		b & -a
	\end{pmatrix}
\ . 
\end{align*}
And thanks to Lemma \ref{lm:hopfdifflocalform}, we can write the Hopf differential of $N$ as
\begin{align*}
	\Phi = \tfrac{1}{2}(a-ib)dz^2
\ . 
\end{align*}
We then recall the definition of the derivative of a tensor,
\[ (\nabla_X\II_0)(Y,Z) = X(\II_0(Y,Z))-\II_0(\nabla_XY,Z)-\II_0(Y,\nabla_XZ)\ ,\]
so that we may explicit the equation in the basis. In order to do that we need the Christoffel symbols for the metric $g=e^{2w}(dx^2+dy^2)$.\\
Let $\Gamma^k_{ij}$ be such that
\begin{align*}
\nabla_{\partial_i}\partial_j=\Gamma^k_{ij}\partial_k\ . 
\end{align*}
Since $\langle \partial_x,\partial_x\rangle=\langle \partial_y,\partial_y\rangle=e^{2w}$ and $\langle \partial_x,\partial_y\rangle=0$, the compatibility with the metric yields
\begin{align*}
\Gamma^1_{11}e^{2w}=\langle \nabla_{\partial_x}\partial_x,\partial_x\rangle
&=\frac12\partial_x\langle \partial_x,\partial_x\rangle=
e^{2w}w_x\\
\Gamma^2_{11}e^{2w}=
\langle \nabla_{\partial_x}\partial_x,\partial_y\rangle
&=-\frac12\partial_y\langle \partial_x,\partial_x\rangle
=-e^{2w}w_y,
\end{align*}
hence $\Gamma^1_{11}=w_x$ and $\Gamma^2_{11}=-w_y$. Similarly,
\begin{align*}
\Gamma^1_{12}e^{2w}=
\langle \nabla_{\partial_x}\partial_y,\partial_x\rangle
&=\frac12\partial_y\langle \partial_x,\partial_x\rangle
=e^{2w}w_y,\\
\Gamma^2_{12}e^{2w}=
\langle \nabla_{\partial_x}\partial_y,\partial_y\rangle
&=\frac12\partial_x\langle \partial_y,\partial_y\rangle
=e^{2w}w_x.
\end{align*}
so $\Gamma^1_{12}=w_y$ and $\Gamma^2_{12}=w_x$. Since the connection is torsion free, $\Gamma^k_{21}=\Gamma^k_{12}$. Finally,
\begin{align*}
\Gamma^1_{22}e^{2w}=
\langle \nabla_{\partial_y}\partial_y,\partial_x\rangle
&=-\frac12\partial_x\langle \partial_y,\partial_y\rangle
=-e^{2w}w_x,\\
\Gamma^2_{22}e^{2w}=
\langle \nabla_{\partial_y}\partial_y,\partial_y\rangle
&=\frac12\partial_y\langle \partial_y,\partial_y\rangle
=e^{2w}w_y.
\end{align*}
so $\Gamma^1_{22}=-w_x$ and $\Gamma^2_{22}=w_y$. Therefore
\begin{align*}
\nabla_{\partial_x}\partial_x&=w_x\partial_x-w_y\partial_y\ ,\\
\nabla_{\partial_x}\partial_y&=w_y\partial_x+w_x\partial_y=\nabla_{\partial_y}\partial_x\ ,\\
\nabla_{\partial_y}\partial_y&=-w_x\partial_x+w_y\partial_y\ .
\end{align*}
We observe that $\II_0$ is symmetric, and therefore $\nabla \II_0$ is symmetric in the last two entries, therefore the Codazzi equations in the basis $(\partial_x,\partial_y)$ reduce to
\begin{align*}
	\begin{cases}
		(\nabla_{\partial_x}\II_0)(\partial_x,\partial_y)
		&=(\nabla_{\partial_y}\II_0)(\partial_x,\partial_x),\\
		(\nabla_{\partial_x}\II_0)(\partial_y,\partial_y)
		&=(\nabla_{\partial_y}\II_0)(\partial_x,\partial_y).
	\end{cases}
\end{align*}
We can calculate $(\nabla_{\partial_x}\II_0)(\partial_x,\partial_y)$ and $(\nabla_{\partial_y}\II_0)(\partial_x,\partial_x)$,
\begin{align*}
(\nabla_{\partial_x}\II_0)(\partial_y,\partial_x)
&=\partial_x\bigl(\II_0(\partial_y,\partial_x)\bigr)
-\II_0(\nabla_{\partial_x}\partial_y,\partial_x)
-\II_0(\partial_y,\nabla_{\partial_x}\partial_x)\\
&=\partial_x b
-\II_0(w_y\partial_x+w_x\partial_y,\partial_x)
-\II_0(\partial_y,w_x\partial_x-w_y\partial_y)\\
&=b_x-(w_ya+w_xb)-(w_xb+w_ya)\\
&=b_x-2w_xb-2w_ya,\ \\
(\nabla_{\partial_y}\II_0)(\partial_x,\partial_x)
&=\partial_y\bigl(\II_0(\partial_x,\partial_x)\bigr)
-\II_0(\nabla_{\partial_y}\partial_x,\partial_x)
-\II_0(\partial_x,\nabla_{\partial_y}\partial_x)\\
&=\partial_y a
-\II_0(w_y\partial_x+w_x\partial_y,\partial_x)
-\II_0(\partial_x,w_y\partial_x+w_x\partial_y)\\
&=a_y-(w_ya+w_xb)-(w_ya+w_xb)\\
&=a_y-2w_ya-2w_xb.
\end{align*}
Hence
\begin{align*}
	(\nabla_{\partial_x}\II_0)(\partial_x,\partial_y)=(\nabla_{\partial_y}\II_0)(\partial_x,\partial_x) \iff b_x = a_y
\ . 
\end{align*}
Developing the calculations in the same way we obtain the equivalence
\begin{align*}
	(\nabla_{\partial_x}\II_0)(\partial_y,\partial_y)=(\nabla_{\partial_y}\II_0)(\partial_x,\partial_y) \iff b_y = -a_x\ .
\end{align*}
So that the Codazzi equations reduce to
\begin{align*}
	\begin{cases}
		a_x = (-b)_y\\
		a_y = -(-b)_x 
	\end{cases}
\ , 
\end{align*}
which are the Cauchy-Riemann equations for $\varphi = a-ib$. We can summarize the content of this section in a statement of a theorem
\begin{thm}\label{th:codhophol}
	Let $S$ be a constant mean curvature surface, and let $N:\mathbb{D}\to\mathbb{H}$ be its Gauss map. Then the Hopf differential of $N$ is holomorphic if and only if the second fundamental form $\II$ of $S$ satisfies the Codazzi equations.
\end{thm}
\subsection{A different form for Gauss' equation}\label{sc:geq}\mbox{}\\
We shall now prove the equivalence of Gauss' equation with a certain PDE on $\mathbb{D}$, which we will derive with respect to the Euclidean backgroun metric $|dz|^2$ and then rewrite it with respect to the hyperbolic metric.\\

Let $\I=e^{2w}|dz|^2$, and let $(x,y)$ be the corresponding isothermal coordinates, with associated complex coordinate $z=x+iy$. It is clear that
\[
\det(g_{ij})=e^{4w}
\ \ .
\]
Let moreover
\[
\II=h_{ij}\,dx^i dx^j
\quad\text{and}\quad
\Phi=\Hopf(N)=\varphi\,dz^2,
\]
where $N$ is the Gauss map of the immersion $S\hookrightarrow \M$.
Recall Gauss' equation:
\[
K=-\det_\I(\II)=-\frac{\det(h_{ij})}{\det(g_{ij})}
\ \ .
\]
We already know the expression of $\det (g_{ij}) $, we need to rewrite $\det(h_{ij})$.
\begin{align*}
	\det(h_{ij})
	&= h_{11}h_{22}-h_{12}^2\\
	&= \tfrac{1}{4}\left[(h_{11}+h_{22})^2-(h_{11}-h_{22})^2\right]-h_{12}^2\\
	\commb{\qref{eq:gausspde1}}\hspace{5mm}
	&= \tfrac{1}{4}\left[(2e^{2w})^2-(2\Re(\varphi))^2\right]-\Im(\varphi)^2\\
	&= e^{4w}-\Re(\varphi)^2-\Im(\varphi)^2\\
	&= e^{4w}-|\varphi|^2
\ \ .
\end{align*}
Now we recall that
\[
K = -e^{-2w}\Delta w
\ \ .
\]
So, substituting in Gauss' equation, we obtain
\begin{align*}
	-e^{-2w}\Delta w
	&=
	-\frac{e^{4w}-|\varphi|^2}{e^{4w}}\\
	&=
	-1+|\varphi|^2e^{-4w}
\ \ ,
\end{align*}
and therefore
\begin{align}\label{eq:gausspde2}
	\Delta w
	&=
	e^{2w}-|\varphi|^2e^{-2w}
\ \ .
\end{align}
This is the Euclidean form of the equation, which will be studied in the later sections.\\

For completeness, let us now rewrite the same equation with respect to the hyperbolic background metric
\[
ds_p^2=\frac{4}{(1-|z|^2)^2}|dz|^2.
\]
If the first fundamental form is expressed by
\[
\I=e^{2\tilde w}ds_p^2,
\]
then
\[
e^{2w}|dz|^2=e^{2\tilde w}\frac{4}{(1-|z|^2)^2}|dz|^2,
\]
hence
\begin{align}\label{eq:gausspde3}
	w=\tilde w+\log\left(\frac{2}{1-|z|^2}\right)
\ \ .
\end{align}
Moreover,
\[
\Delta_p=\frac{(1-|z|^2)^2}{4}\Delta
\]
and
\[
\Delta \log\left(\frac{2}{1-|z|^2}\right)=\frac{4}{(1-|z|^2)^2}.
\]
Therefore
\begin{align*}
	\Delta w
	&=
	\Delta \tilde w
	+
	\Delta \log\left(\frac{2}{1-|z|^2}\right)\\
	&=
	\frac{4}{(1-|z|^2)^2}\Delta_p\tilde w
	+
	\frac{4}{(1-|z|^2)^2}\\
	&=
	\frac{4}{(1-|z|^2)^2}\left(\Delta_p\tilde w+1\right)
\ \ .
\end{align*}
On the other hand, by \eqref{eq:gausspde3},
\[
e^{2w}=\frac{4}{(1-|z|^2)^2}e^{2\tilde w}.
\]
Also, if $\|\Phi\|$ denotes the norm of $\Phi$ with respect to the Poincaré metric $ds_p^2$, then
\[
\|\Phi\|^2=\frac{(1-|z|^2)^4}{4^2}|\varphi|^2,
\]
that is,
\[
|\varphi|^2=\frac{16}{(1-|z|^2)^4}\|\Phi\|^2.
\]
Hence
\[
|\varphi|^2e^{-2w}
=
\frac{4}{(1-|z|^2)^2}\|\Phi\|^2e^{-2\tilde w}.
\]
Substituting these expressions into \eqref{eq:gausspde2}, we get
\begin{align*}
	\frac{4}{(1-|z|^2)^2}\left(\Delta_p\tilde w+1\right)
	&=
	\frac{4}{(1-|z|^2)^2}e^{2\tilde w}
	-
	\frac{4}{(1-|z|^2)^2}\|\Phi\|^2e^{-2\tilde w}
\ \ ,
\end{align*}
and therefore
\begin{align}\label{eq:gausspde4}
	\Delta_p\tilde w
	&=
	e^{2\tilde w}
	-
	\|\Phi\|^2e^{-2\tilde w}
	-
	1
\ \ .
\end{align}
This is exactly the form of the PDE found in \cite{Wan1992}. Again, it might be useful to summarize this result in a theorem statement.
\begin{thm}\label{thm:gausseqnthm}
	Let $S$ be a constant mean curvature $1$ surface, let $N:\mathbb{D}\to\mathbb{H}$ be its Gauss map, and let $\Phi=\phi\,dz^2$ be its Hopf differential. If the induced metric on $S$ is written as
	\begin{align*}
		\I=e^{2w}ds_{p}^2,
	\end{align*}
	then the Gauss equation for $S$ is equivalent to
	\begin{align*}
		\Delta_{p}w=e^{2w}-\|\Phi\|^2e^{-2w}-1.
	\end{align*}
\end{thm}

\section{CMC surfaces and harmonic maps}

We now introduce the class of surfaces that will be studied throughout this section.
\begin{defn}
	A constant mean curvature cut is a complete entire spacelike surface
	$S \subset \M$ with constant mean curvature $H>0$.
\end{defn}
This is essentially the same setting we exposed in Section \ref{sc:splsrf} if not for the additional condition of having a constant mean curvature.
The first step for classification is the following proposition.
\begin{prop}\label{pr:convfun}
	Let $S$ be a complete spacelike surface in $\M$ with constant mean curvature $H$.
	$S$ can be represented as the graph of an entire convex function
	$x^3=f(x^1,x^2)$.
	The Gaussian curvature of $S$ satisfies
	$K \leq 0$. More precisely, either $K<0$ everywhere or if  $K=0$ at some point then $K\equiv 0$ and the surface is parabolic.
\end{prop}

An immediate consequence of being the graph of a convex function is that the second fundamental form of the surface is positive definite, from which we get that the mean curvature must be positive.\\

We observe now that we may suppose the mean curvature of $S$ to be $1$. In fact, if $\iota:S\hookrightarrow \M$ is an immersion with constant mean curvature $H=k>0$, then we may consider the dilated immersion
\[
\tilde \iota = \mu\iota ,
\]
which has constant mean curvature $\tilde H = \mu^{-1}k$.
Choosing $\mu= k$, we obtain a surface with constant mean curvature $1$.

Therefore, there exists a bijection between complete space-like surfaces with constant mean curvature $k$ and complete space-like surfaces with constant mean curvature $1$. Hence a classification of the latter immediately yields a classification of the former. We may then assume from now on that the surfaces have constant mean curvature $1$.\\


Let us then fix $S$ a hyperbolic CMC cut with mean curvature $1$. The hyperbolic hypothesis means that using the uniformization theorem we can find a conformal diffeomorphism
\[
	\Lambda:\mathbb D \to S.
\]
Let us now consider the Gauss map of  $S$, $u:S\to \mathbb{D}$. By composing with $\Lambda$ read it as a map
\[
	u:\mathbb D \to \mathbb{D}.
\]
The hypothesis on the mean curvature of $S$ being constant lets us apply the Theorem  \ref{th:gaussmapharmiffconsth} and obtain that $u$ is harmonic (we remind that $\mathbb{D}$ is isometric to $\mathbb{H}$ found in the statement of the theorem).\\

Now, we can pull back the metric of $S$ through the  conformal parametrization and express it in terms of  $u$:
\begin{prop}\label{prop:indmetr}
	Suppose that $S$ has constant mean curvature $1$. Then the pulled-back induced metric $g=\Lambda^*(\I)$ of $S$ and the respective Gaussian curvature are given by
	\begin{align*}
		g = \frac{4}{(1-|u|^2)^2}|u_z|^2 |dz|^2, \qquad K = \frac{|u_{\overline z}|^2}{|u_z|^2} - 1
	\ \ .
	\end{align*}
\end{prop}
\begin{proof}
	We redirect the reader to \cite{akutagawa_nishikawa_1990}.
\end{proof}
We now need a claim we will prove later in the section.
\begin{claim}\label{cl:jhupos}
	The Gauss map $u$ satisfies  
	\[J(u) > 0\qquad\text{and}\qquad \mathcal H(u)>0\ .\] 
\end{claim}
The fact that $J(u)>0$ along with $u$ being harmonic lets us apply Corollary \ref{cor:diffeopresthenharm} and conclude that its Hopf differential is a holomorphic quadratic differential. So, if we denote by
\[\Hopf(u)(z) = \varphi(z) dz^2\]
then $\varphi(z) = \rho^2(u(z)) u_z \overline{u}_{\overline{z}} $ is holomorphic. For brevity we set $\Phi:=\Hopf(u)$\\

Let us now prove the claim 
\begin{proof}[Proof of Claim \ref{cl:jhupos}]
	First, by Proposition \ref{prop:indmetr} we have that the metric $g:= \Lambda^*(\I_S)$ is expressed by
	\[g =  \frac{4}{(1-|u|^2)^2}|u_z|^2 |dz|^2\ .\] 
	Then we recall that the metric on $\mathbb{D}$ at a point $v$ is $ds_p^2|_v:= \frac{4}{(1-|v|^2)^2} |dz|^2$.
	So we have
	\begin{align*}
		g &=  \frac{4}{(1-|u|^2)^2} |u_z|^2|dz|^2 \\
		&=  ds_p^2|_u|u_z|^2  \\
		&= ds_p^2|_u \mathcal H(u) 
	\ \ . 
	\end{align*}
	We know that both  $g$ and $ds_p^2$ are Riemannian metrics, so that we obtain $\mathcal H(u)>0$.

	Now, from the second equation of Proposition \ref{prop:indmetr} we have
	\begin{align*}
		K = \frac{|u_{\overline{z}}|^2}{|u_z|^2}-1
	\ \  
	\end{align*}
	so that	
	 \begin{align*}
		 |u_z|^2 K &= |u_{\overline{z}}|^2-|u_z|^2\\
		 &=  -J(u) 
	\ \ . 
	\end{align*}
	Then \[J(u)=-|u_z|^2K>0\ ,\] since $K<0$, which is true by applying Proposition \ref{pr:convfun} and recalling that $S$ is hyperbolic.
\end{proof}
From the proof we can also observe that the metric induced on $S$ can be expressed as $g = \mathcal H ds_p^2 = e^{2w}ds_p^2$.\\

We have proven the following theorem:
\begin{thm}\label{th:pdenecthm}
	Let $S$ be a hyperbolic constant mean curvature cut with mean curvature equal to 1, let $u:\mathbb{D}\to \mathbb{H}$ be the Gauss map of $S$ read through the conformal parametrization $\Lambda$ and $\Phi = \varphi(z)dz^2$ its Hopf differential. Then
	\begin{itemize}[itemsep=0mm]
		\item $\Phi$ is holomorphic as a quadratic differential,
		\item $u$ preserves the orientation,
		\item $w := \frac{1}{2}\log\mathcal{H}(u)$ is a well defined function on $\mathbb{D}$,
		\item $e^{2w}ds_p^2$ is  the metric of $S$ read through the conformal parametrization $\Lambda$,
	\end{itemize}

\end{thm}
\section{Classification of hyperbolic CMC cuts}


In the last section we essentially followed the sequence of constructions
\begin{align}\label{eq:infrel}
S\longmapsto u\longmapsto (\Phi,w)\longmapsto \Phi,
\end{align}
where $S$ is a hyperbolic CMC-$1$ cut in $\M$; $u:\mathbb{D}\to \mathbb{H}$ is the Gauss map of $S$ read through the conformal parametrization obtained by the uniformization theorem; $\Phi$ is the Hopf differential of $u$ and $w=\frac{1}{2}\log\mathcal{H}(u)$ with $\mathcal{H}$ denoting the holomorphic energy of $u$. Furthermore we showed that the involved quantities satisfy the properties in Theorem \ref{th:pdenecthm}\\

As it is, the assignment from the surface $S$ to the holomorphic quadratic differential on $\mathbb{D}$, $S\mapsto \Phi$, is not well defined, because it depends on the choice of the conformal parametrization. This ambiguity disappears after passing to equivalent classes by considering the map
\begin{align*}
	\mathscr{A} : \mathbf{E} \to \mathbf{Q}_{\mathbb{D}}
\ \ , 
\end{align*}
where $\mathbf{E}$ is the space of surfaces $S$ up to isometry and $\mathbf{Q}_{\mathbb{D}}$ is the space of holomorphic quadratic differentials up to change of conformal coordinates. This is in fact justified because we can't distinguish from an intrinsic point of view two isometric surfaces, while a quadratic differential compatible with the complex structure should be intended independently from its local representation.\\
So we can prove the theorem:
\begin{thm}\label{th:class}
	The map $\mathscr{A}$ is well defined and bijective.
\end{thm}
The proof of this theorem requires an existence and uniqueness result for the conformal factor of the metric induced on $S$, expressed in terms of a holomorphic quadratic differential which will later be identified with the Hopf differential of the Gauss map. We state this result here and provide proof to the next sections.
\begin{claim}\label{cl:existenceuniquew}
	Let $\Phi$ be a holomorphic quadratic differential on $\mathbb{D}$. Then there exists a unique smooth function $w:\mathbb{D}\to\mathbb{R}$ such that
	\begin{align*}
		\Delta_p w = e^{2w}-\|\Phi\|^2e^{-2w}-1,
	\end{align*}
	the metric $e^{2w}ds_p^2$ is complete and $e^{2w}-\|\Phi\|^2e^{-2w}>0$.
\end{claim}
\begin{proof}[Proof of Theorem \ref{th:class}]\mbox{}\vspace*{-1.5mm}
\begin{enumerate}[noitemsep,leftmargin=*,label=\textbf{\arabic*.}]
\item Well-definedness:\\
We just observe that by choosing a different $S'\in [S]$ ultimately results to the same map $u$ post-composed with an isometry of the hyperbolic disk $\mathbb{D}$. The Hopf differential associated to $u$ remains unchanged under this transformation.
\item Surjectivity:\\
Let $[\Phi]\in \mathbf{Q}_{\mathbb{D}}$ and choose a representative
\[
\Phi=\varphi(z)\,dz^2.
\]
We reconstruct the surface $S$ starting from $\Phi$ using the following diagrams
\begin{center}	
\begin{tikzpicture}[
	every node/.style={inner sep=2pt},
	lbl/.style={font=\scriptsize}
]
\node (phi) at (0,0) {$\Phi$};
\node (normphi) at (1.6,0) {$\lVert \Phi \rVert$};
\node (w) at (3.2,0) {$w$};

\node (hij) at (5.2,0) {$(h_{ij})_{ij}$};
\node (metric) at (3.2,-1.6) {$e^{2w}ds_p^2$};
\node (gij) at (5.2,-1.6) {$(g_{ij})_{ij}$};

\node[draw,circle] (gc) at (6.8,-0.8) {\tiny \textsf{G.C.}};
\node (S) at (8.2,-0.8) {$S$};

\draw[|->] (phi) -- node[above,lbl] {\astr{1}} (normphi);
\draw[|->] (normphi) -- node[above,lbl] {\astr{2}} (w);

\draw[|->] (w) -- node[above,lbl] {\astr{5}} (hij);
\draw[|->] (w) -- node[right,lbl] {\astr{3}} (metric);
\draw[|->] (metric) -- node[above,lbl] {\astr{4}} (gij);

\draw[|-] (hij) -- node[above,lbl] {} (gc);

\end{tikzpicture}.
\end{center}

\begin{itemize}[noitemsep]
	\item[\astr{1}] Starting from the holomorphic quadratic differential $\Phi$, we compute its norm $\|\Phi\|$ with respect to the Poincaré metric.
	\item[\astr{2}] We recover the function $w$ using Claim \ref{cl:existenceuniquew}.
	\item[\astr{3}] We define the bilinear form
	\[
		\I=e^{2\tilde w}|dz|^2 = e^{2w}ds_p^2\ .
	\]
	Which, by the choice of $w$, it defines a complete metric on $\mathbb{D}$. 
	\item[\astr{4}] If the first fundamental form is
	\[
	\I=e^{2\tilde w}(dx^2+dy^2),
	\]
	then its coefficients in the coordinates $(x,y)$ are
	\[
	(g_{ij})_{ij}=e^{2\tilde w}
	\begin{pmatrix}
	1&0\\
	0&1
	\end{pmatrix}.
	\]
	\item[\astr{5}] We now define a candidate second fundamental form
	\[
	\II = h_{ij}dx^i dx^j
	\]
	by requiring that
	\begin{equation}\label{eq:class1}
		\begin{cases}
		\frac{1}{2}(h_{11}+h_{22})= e^{2\tilde w}\\
		h_{11}-h_{22} = 2\Re(\varphi)\\
		h_{12} = -\Im(\varphi)\ ,
		\end{cases}
	\ \ 
	\end{equation}
	which are the equations we found in \eqref{eq:gausspde1}. This is a linear system which can be easily solved to find $(h_{ij})_{ij}$.
	\item[\astr{6}] We claim that the pair $(\I,\II)$ satisfies the Gauss-Codazzi equations.\\

	For the Codazzi equations, we may recognize that the statement of Theorem \ref{th:codhophol} holds not only if $\Phi$ is the Hopf differential of a Gauss map, but also by simply assuming that the relation \eqref{eq:class1} holds.\\

	Regarding Gauss' equation, by Theorem \ref{thm:gausseqnthm}, it is equivalent to $w$ solving the equation
	\[
	\Delta_p w = e^{2w}-\|\Phi\|^2e^{-2w}-1\ ,
	\]
	which we assumed to be true by claim.\\

	Hence $\I$ and$\II$ satisfy the Gauss-Codazzi equations and therefore we can apply Bonnet's Theorem \ref{th:bonnet}, finding for every point of $\mathbb{D}$ a local immersion in $\M$, whose first and second fundamental forms are $\I$ and $\II$. Since $\mathbb{D}$ is simply connected, we can glue these local immersions to a global one
	\[
	\iota:\mathbb{D}\to \M,
	\]
	unique up to post-composition with an isometry of $\M$.
	

	The induced metric of $\iota$ is $\I=e^{2w}ds_p^2$, hence complete. Moreover the mean curvature is identically $1$, for how we constructed the forms $\I$ and $\II$. Therefore the image surface is a complete spacelike surface in $\M$ with constant mean curvature $1$. Then, by Proposition \ref{pr:convfun}, it is an entire graph, hence a hyperbolic CMC-$1$ cut.
\end{itemize}
Finally, by construction, the Hopf differential of the Gauss map of the surface we just found is $\Phi$. Therefore
\[
\mathscr A([S])=[\Phi],
\]
and $\mathscr A$ is surjective.
\item Injectivity:\\
Let $[S_1],[S_2]\in \mathbf{E}$ and assume that
\[
\mathscr A([S_1])=\mathscr A([S_2]).
\]
We want to show that $[S_1]=[S_2]$.

Let us choose some conformal parametrizations
\[
\Lambda_i:\mathbb{D}\to S_i,
\qquad i=1,2,
\]
and define
\[
\Phi_i=\varphi_i(z)\,dz^2
\]
the corresponding quadratic differentials of their Gauss maps. Since we assumed that 
$
\mathscr A([S_1])=\mathscr A([S_2])
$, up to a change of parameter we may assume that
\[
\Phi_1=\Phi_2=:\Phi.
\]

Then we define $w_i$ be the function associated to $S_i$ as in Theorem \ref{th:pdenecthm}. Again from the theorem, we have that
\[
\I_i=e^{2w_i}ds_p^2
\]
is the first fundamental form of $S_i$ read through $\Lambda_i$. Since both surfaces have Hopf differential $\Phi$, the functions $w_1$ and $w_2$ solve the same equation
\[
\Delta_p w = e^{2w}-\|\Phi\|^2e^{-2w}-1,
\]
and the metrics
\[
e^{2w_1}ds_p^2,
\qquad
e^{2w_2}ds_p^2
\]
are complete. By the Claim \ref{cl:existenceuniquew}, it follows that
\[
w_1=w_2.
\]
Hence $S_1$ and $S_2$ have the same first fundamental form. Moreover, the second fundamental forms are determined by $w$ and $\Phi$ through the relations 
\begin{align*}
	\begin{cases}
	\frac{1}{2}(h_{11}+h_{22})&=e^{2\tilde w},\\
	h_{11}-h_{22}&=2\Re(\varphi),\\
	h_{12}&=-\Im(\varphi),
	\end{cases}
\end{align*}
so that they also coincide.

Therefore the two surfaces have the same first and second fundamental forms on $\mathbb{D}$ and by applying Bonnet's Theorem \ref{th:bonnet}, the corresponding immersions differ by a post-composition with an isometry of $\M$. In particular, $S_1$ and $S_2$ are isometric, hence
\[
[S_1]=[S_2]
\]
in $\mathbf{E}$.
Which proves that $\mathscr A$ is injective.
\end{enumerate}
\end{proof}



\section{Uniqueness for the PDE}
We now begin the proof of Claim \ref{cl:existenceuniquew}. Recall that, if $\Phi$ is a holomorphic quadratic differential on $\mathbb{D}$ and we set
\[
q=\|\Phi\|,
\]
the claim asserts the existence and uniqueness of a smooth function $w:\mathbb{D}\to\mathbb{R}$ satisfying
\begin{align}\label{eq:box}
	\Delta_p w = e^{2w}-q^2e^{-2w}-1,
\end{align}
such that the metric $e^{2w}ds_p^2$ is complete and
\begin{align}\label{eq:box2}
	e^{2w}-q^2e^{-2w}>0
\end{align}
on $\mathbb{D}$.
In this section we prove uniqueness.
\subsection{A generalized maximum principle}
Let $M$ be a complete Riemannian manifold of dimension $n$, and fix a point $p\in M$. Assume moreover that $M$ is simply connected and has nonpositive sectional curvature. This assumption is stronger than what is strictly needed in this section, but greatly simplifies the discussion and is satisfied in the case of a complete space-like surface of $\M$. These hypotheses allow us to apply the Cartan-Hadamard theorem, so that the exponential map $\exp_p$ is a diffeomorphism. In particular, this tells us that for every $x\in M$ there exists a unique geodesic joining $p$ to $x$, and that $d(\cdot ,\cdot )$ is smooth on $M\setminus \{x=y\} $.\\

We need now to define an important quantity we will refer to multiple times later. Let us fix a point $x\in M$ and let $\sigma$ be the unique unit speed geodesic that joins $x$ to $p$. In the following expression we also denote with $l$ the length of $\sigma$, or in other terms $l=d(x,p)$. Let $\sigma'(t)$ be the velocity vector of $\sigma$, and denote with $\Ric(\sigma'(t))$ the Ricci curvature along $\sigma$. We then define
\begin{align}\label{eq:intromaxprin1}
	K(x) = \min_{k\in [0,l]}  \left[ \frac{n-1}{l-k} - \frac{1}{(l-k)^2} \int_k^l (t-k)^2\Ric(\sigma'(t))dt\right]
\ \ .
\end{align}
We now claim that, if $r(x) \geq 1$ and the Ricci curvature is bounded from below, then $K(x)$ is bounded from above by a constant independent of $x$. Indeed, if we assume that for every unit vector $v$, $\Ric(v)\geq \mathscr{S}$, then
\begin{align*}
	\frac{1}{(l-k)^2} \int_k^l (t-k)^2\Ric(\sigma'(t)) \,dt
	\geq
	\frac{1}{(l-k)^2} \int_k^l (t-k)^2\mathscr{S}\,dt
	=
	\frac{(l-k)}{3}\mathscr{S}
\ \ .
\end{align*}
Therefore in the argument of $\min$ in \eqref{eq:intromaxprin1} we have
\begin{align*}
	\frac{n-1}{l-k} - \frac{1}{(l-k)^2} \int_k^l (t-k)^2\Ric(\sigma'(t)) \,dt
	\leq
	\frac{n-1}{l-k}-\frac{(l-k)}{3}\mathscr{S}
\ \ .
\end{align*}
Since $r(x)=l\geq 1$, if we choose $k=l-1$ we get
\begin{align*}
	K(x)\leq n-1-\frac{\mathscr{S}}{3}
\ \ .
\end{align*}
Which proves our claim.\\

The next lemma, due to Yau, provides the estimate for the distance function that will be used in the proof of Theorem \ref{th:maxprin}.
\begin{lemma}\label{lm:distest}
With the notation introduced above, for every $x\in M\setminus\{p\}$ such that $r(x)\geq 1$ we have
\begin{align*}
	\Delta r(x)\leq K(x)
	\ .
\end{align*}
\end{lemma}
\begin{proof}
Since $M$ is complete, simply connected, and has nonpositive curvature, the Cartan--Hadamard theorem implies that
\[
\exp_p:T_pM\to M
\]
is a diffeomorphism. In particular, $r$ is smooth on $M\setminus\{p\}$.\\

Let us fix an orthonormal basis $(e_1,e_2)$ of $T_pM$, and consider the polar coordinates
\[
\Phi(r,\theta)=\exp_p\bigl(r\cos\theta\,e_1+r\sin\theta\,e_2\bigr).
\]
We denote by $\partial_r$ and $\partial_\theta$ the corresponding coordinate vector fields. It is a well known fact (see for example \cite{do_carmo_2016}) that using these coordinates we can express the metric as 
\[
g = dr^2 +u(r,\theta)^2\, d\theta^2
\]
for a certain $u$, and that the following equation holds
\begin{align}\label{eq:boundr0}
	u_{rr} = -K(r,\theta) u\ .
\end{align}
Now we observe that, by definition 
\begin{align}\label{eq:boundr1}
	\nabla r = \left( dr \right)^\sharp = \partial_r
\ . 
\end{align}
Let us fix $E_1$, $E_2$ an orthonormal frame. A suitable choice is $E_1 = \partial_r$ and $E_2 = \frac{1}{u} \partial_\theta$.
Then we have
\begin{align}\label{eq:boundr2}
	\Delta r &= \nabla\cdot \nabla r\notag\\
	\commb{\qref{eq:boundr1}}\hspace{4mm}&=  \nabla\cdot \partial_r \notag\\
	&=  \left< \nabla_{E_1} \partial_r, E_1 \right> + \left< \nabla_{E_2}\partial_r, E_2 \right> \notag\\
	&=  \left<\nabla_{\partial_r}\partial_r,\partial_r \right> + \frac{1}{u^2} \left<\nabla_{\partial_\theta} \partial_r, \partial_\theta \right> 
\ . 
\end{align}
Now we observe that since at $\theta$ fixed, the map  $r \mapsto \Phi(r,\theta)$ is a geodesic, we have $\nabla_{\partial_r}\partial_r = 0$. Then substituting in \eqref{eq:boundr2} we obtain
\begin{align}\label{eq:boundr3}
	\Delta r = \frac{1}{u^2}\left< \nabla_{\partial_\theta}\partial_r,\partial_\theta \right>
\ . 
\end{align}
We make two more observations. First we recall that
\begin{align*}
	|\partial_\theta|^2 = \left< \partial_\theta,\partial_\theta\right> = u^2
\ . 
\end{align*}
And then we observe that
\begin{align*}
	\partial_r |\partial_\theta|^2
	&= \partial_r(u^2) \\
	&= 2u \partial_r(u)\\
	&= 2u u_r
\ . 
\end{align*}
Continuing in \eqref{eq:boundr3}, we obtain
\begin{align*}
	\Delta r &= \frac{1}{u^2}\left< \nabla_{\partial_\theta} \partial_r, \partial_\theta \right> \\
	\commb{$\nabla$ Levi-Civita}\hspace{4mm} &= \frac{1}{u^2}\left< \nabla_{\partial_r} \partial_\theta, \partial_\theta \right> \\
	&= \frac{1}{2u^2} \partial_r \left<\partial_\theta,\partial_\theta \right> \\
	&=  \frac{1}{2u^2} \partial_r |\partial_\theta|^2 \\
	&= \frac{1}{2u^2} 2u u_r  \\
	&= \frac{u_r}{u}
\ . 
\end{align*}
Now set $f= \dfrac{u_r}{u} = \Delta r$. We have
\begin{align*}
	f_r &= \partial_r \left( \frac{u_r}{u} \right) \\
	&= \frac{u_{rr}}{u} - \frac{u_r^2}{u^2} \\
	&=  \frac{u_{rr}}{u} - f^2 \\
	\commb{\qref{eq:boundr0}}\hspace{5mm}&= -K -f^2
\ . 
\end{align*}

Since the Gaussian curvature is bounded from below, there exists a constant $A_0\ge 0$ such that
\[
K\ge -A_0.
\]
Hence
\begin{align}\label{eq:boundr3a}
	f_r\le A_0-f^2.
\ . 
\end{align}
Moreover, since $f$ is continuous on the compact set $\{r=1\}$, we can choose a constant $A\ge A_0$ sufficiently large such that 
\[
f(1,\theta)\le \sqrt{A}+1
\qquad\text{for every }\theta.
\]
Therefore
\[
f_r\le A-f^2
\qquad\text{and}\qquad
f(1,\theta)\le \sqrt{A}+1
\]
for every $\theta$.

We claim that
\[
f(r,\theta)\le \sqrt{A}+1
\qquad\text{for every }r\ge 1\text{ and every }\theta.
\]
Let us fix a $\theta$. If this were false, there would exist an $r_0>1$ such that for every $r\in [1,r_0)$
\begin{align}\label{eq:boundr4}
	f(r_0,\theta)=\sqrt{A}+1,
	\qquad
	f(r,\theta)<\sqrt{A}+1 
\ . 
\end{align}
Since $r_0$ is the first value of $r$ such that $f(r,\theta)=\sqrt A+1$, for every $r<r_0$ sufficiently close to $r_0$ we have
\[
f(r,\theta)<f(r_0,\theta).
\]
Recall that 
\begin{align*}
	\partial_r f(r,\theta) = \lim_{r\to r_0^-} \frac{f(r,\theta)-f(r_0,\theta)}{r-r_0} 
\ . 
\end{align*}
But inside the limit, both the numerator and the denominator are positive for $r$ close enough to $r_0$, due to \eqref{eq:boundr4}.
Hence 
\begin{align*}
	f_r(r_0,\theta) \geq 0
\ . 
\end{align*}
On the other hand, from \ref{eq:boundr3a} we get
\[
f_r(r_0,\theta)
\le
A-f(r_0,\theta)^2
=
A-(\sqrt{A}+1)^2
<
0,
\]
which is a contradiction. Hence
\[
f(r,\theta)\le \sqrt{A}+1
\]
for every $r\ge 1$ and every $\theta$.

Finally we conclude the proof by recalling that $f=\Delta r$, so that
\[
\Delta r(x)\le \sqrt{A}+1
\]
for every $x$ such that $r(x)\ge 1$.
\end{proof}




















We are ready to state and prove the maximum principle.
\begin{thm}\label{th:maxprin}
Let $M$ be a complete, simply connected Riemannian manifold of nonpositive sectional curvature, with Ricci curvature bounded from below. Let $f\in C^2(M)$ be bounded from above. Fix $p\in M$, and let $r(x)=d(p,x)$. Then there exist a constant $C>0$ and a sequence $(x_k)_k\subset M$ such that
\[
f(x_k)\to \sup_M f,\qquad |\nabla f|(x_k)\le \frac{C}{k},\qquad \Delta f(x_k)\le \frac{C}{k}.
\]
In particular,
\[
\limsup_{k\to\infty}\Delta f(x_k)\le 0.
\]
\end{thm}
\begin{proof}
	Let us define the auxiliary functions $u(x) = (f(x)-f(p)+1) $ and $L(x) = \log(r^2(x)+2)$. Then let us fix a positive integer $k$ and define the function $g$ as
	\begin{equation*}
		g (x) = \frac{f(x)-f(p)+1}{(\log(r^2(x) +2)^{\frac{1}{k}}} = \frac{u(x)}{L(x)^\frac{1}{k}}
		\ \ .
	\end{equation*} 
	Let us observe that $g(p) = \frac{1}{\log(2)^\frac{1}{k}}>0$ and that $\lim_{r(x) \to \infty}g(x) = 0$. Since $g$ is continuous, this means that it attains its maximum at a certain point $x_k$.\\

	First, let us show that we can assume that $r(x_k) \to  \infty$. If that is not the case, up to taking a subsequence, $r(x_k)$ is bounded.
	Then we can find a closed geodesic ball $\overline{B}(p;\rho)$ with $\rho>0$ that contains $x_k$ for every $k$ sufficiently big. Since $M$ is complete, the Hopf Rinow theorem implies that $\overline{B}(p;\rho)$ is compact. Therefore, after passing to a subsequence, we may assume that 
	\begin{align*}
		x_k \to x_0
	\ \ .
	\end{align*}
	But we know that $f(x_k) \to \sup f$ and that $ f$ is continuous, which implies that $x_0$ is a maximum point.\\
	This entails that 
	\[ \nabla f(x_0) = 0\qquad \text{ and } \qquad \Delta f(x_0) \leq 0\ \ .\]
	Which verifies trivially the thesis by setting $x_k \equiv x_0$. Thus we assume, from now on, that $r(x_k)\to \infty$.\\

	Let us prove first that
	\begin{align}\label{eq:boundprin13}
		\limsup_{k\to \infty}f(x_k) = \sup_M f
	\ \ .
	\end{align}
	Assume it by contradiction to be false. Then by definition of $\sup$ and $\limsup$ there exists a $\delta > 0$ and a $ \bar{x}\in M$ such that for $k$ sufficiently big,
	\begin{align}\label{eq:boundprin12}
		f(x_k)< f(\bar x)-\delta\ , \qquad \text{equivalently},\qquad f(x_k)+\delta < f(\bar x)
	\ \ .
	\end{align}
	Since we assumed that $r(x_k) \overset{k\to \infty}{\longrightarrow} \infty$, then we have, for $k$ sufficiently large, $r(x_k) > r(\bar x) $ and $f(\bar x) > f(x_k)$ so that
	\begin{align*}
		g(\bar x) = \frac{f(\bar x)-f(p)+1}{(\log(r^2(\bar x) +2)^{\frac{1}{k}}} > \frac{f(x_k )-f(p)+1}{(\log(r^2(x_k) +2)^{\frac{1}{k}}} = g(x_k) 
	\ \ .
	\end{align*}
	Which contradicts the maximality of $x_k$.
	It is obvious that, up to taking a subsequence, we can assume $f(x_k) \to  \sup_M f$.\\

	Let us now focus on the two estimates. As we already mentioned, the Cartan-Hadamard $M$ theorem guarantees that the function $d(p,x)^2 = r(x)^2$ is differentiable in $x_k$.\\
	We can easily calculate the gradient of $g$,
	\[
	\nabla g(x)
	=
	\frac{\nabla f(x)}{\log(r(x)^2+2)^{1/k}}
	-
	\frac{2(f(x)-f(p)+1)r(x)}{k(r(x)^2+2)\log(r(x)^2+2)^{1+1/k}}\nabla r(x)
	\ \ .\]
	Since $x_k$ is a maximum point of $g_k$, we have $\nabla g(x_k)=0$. Hence
	\begin{align}
		\nabla f(x_k)
		&=
		\frac{2(f(x_k)-f(p)+1)\,r(x_k)}{k\,(r^2(x_k)+2)\,\log(r^2(x_k)+2)}\nabla r(x_k)\ ,\label{eq:boundprin9}\\
		|\nabla f|(x_k)
		&=
		\frac{2(f(x_k)-f(p)+1)\,r(x_k)}{k\,(r^2(x_k)+2)\,\log(r^2(x_k)+2)}
		\ , \label{eq:boundprin8}
	\end{align}
	where we used $|\nabla r(x_k)|=1$.\\
	We then observe, since $f$ is bounded from above, also $u$ is
	\[ u(x) = ( f(x)-f(p)+1) \leq \sup_M f - f(p) +1  =: A.\]
	Also, it is not difficult to show that the real function satisfies 
	\begin{align}\label{eq:boundprin14}
		\frac{2x}{ (x^2+2) \log(x^2+2)}\leq 2\ ,
	\ \ .
	\end{align}
	 so that by substituting in \eqref{eq:boundprin8} we get the desired estimate 
	\[|\nabla f| (x_k) \leq \frac{2A}{k}\ \ .\] 
	Let us now focus on the estimate on the Laplacian. We can find the expression of the Laplacian of $g=u /L^{\frac{1}{k}}$. First we observe that
	\begin{align}\label{eq:boundprindeltag1}
		\Delta g &= \nabla\cdot \nabla \left( \frac{u}{L^{\frac{1}{k}}} \right)\notag \\
				 &= \nabla \cdot \left( \nabla u L^{-\frac{1}{k}} + u \nabla (L^{-\frac{1}{k}})  \right)   \notag\\
				 &=\frac{\Delta u}{ L^{\frac{1}{k}}} +2 \left< \nabla u, \nabla (L^{-\frac{1}{k}})  \right> + u \Delta(L^{-\frac{1}{k}})
	\ \ ,
	\end{align}
	then we calculate the involved quantities.
	First,	
	\begin{align*}
		\nabla u&= \nabla f,\qquad
		\nabla L = \frac{2r\nabla r}{r^2 +2},\qquad
		\nabla \left( \frac{1}{r^2+2} \right) = -\frac{2r\nabla r}{(r^2+2)^2},\\
		\nabla \left( L^{-\frac{1}{k}} \right)  &= -\frac{1}{k} \frac{\nabla L }{L^{\frac{1}{k}+1}}
		= -\frac{1}{k} \frac{2r\nabla r}{(r^2+2)L^{\frac{1}{k}+1}},\\
		\nabla \left( L^{-\frac{1}{k}-1} \right) &= \left( -\frac{1}{k}-1 \right)\frac{\nabla L}{L^{\frac{1}{k}+2}}
		= -\frac{k+1}{k}\frac{2r\nabla r}{(r^2+2)L^{\frac{1}{k}+2}},\\
		\Delta L &= \nabla \cdot \left( \frac{2r \nabla r}{r^2+2} \right)
		= \frac{2|\nabla r|^2}{r^2+2}+ \frac{2 r \Delta r}{r^2+2}- \frac{4 r^2 |\nabla r|^2}{(r^2+2)^2}
		\ \ ,
	\end{align*}
	and then
	\begin{align*}
		\Delta \left( L^{-\frac{1}{k}} \right) &= \nabla \cdot \nabla L^{-\frac{1}{k}} \\
											   &= -\frac{1}{k}\nabla \cdot \left( \frac{\nabla L}{L^{\frac{1}{k}+1}} \right) \\
		&= -\frac{1}{k} \frac{\Delta L}{L^{\frac{1}{k}+1}}-\frac{1}{k} \left<\nabla L, \nabla \left( L^{-\frac{1}{k}-1} \right)  \right>\\
		&= -\frac{1}{k} \left( \frac{2|\nabla r|^2}{r^2+2} +
		\frac{2r \Delta r}{r^2+2} -
		\frac{4r^2 |\nabla r|^2}{(r^2+2)^2}\right)
		\frac{1}{L^{\frac{1}{k}+1}}
		+
		\frac{k+1}{k^2}\left<\frac{2r \nabla r}{r^2+2}, \frac{2r\nabla r}{(r^2+2)L^{\frac{1}{k}+2}} \right>\\
		&= -\frac{2}{k} \frac{1}{(r^2+2)L^{\frac{1}{k}+1}}
		- \frac{2}{k} \frac{r\Delta r}{(r^2+2) L^{\frac{1}{k}+1}}
		+ \frac{4}{k} \frac{r^2}{(r^2+2)^2 L^{\frac{1}{k}+1}}
		+ \frac{4(k+1)}{k^2} \frac{r^2}{(r^2+2)^2L^{\frac{1}{k}+2}}
		\ .
	\end{align*}
Hence, by substituting in  \eqref{eq:boundprindeltag1} and simplifying, we obtain
	\begin{align*}
		\Delta g &= \frac{\Delta f}{L^{1/k}}
		-\frac{4r}{k(r^2+2)L^{1+1/k}} \langle \nabla f, \nabla r \rangle
		-\frac{2u}{k(r^2+2)L^{1+1/k}} \\
		&\quad
		-\frac{2ur \, \Delta r}{k(r^2+2)L^{1+1/k}}
		+\frac{4ur^2}{k(r^2+2)^2L^{1+1/k}}
		+\frac{4(k+1)ur^2}{k^2(r^2+2)^2L^{2+1/k}}\ .
	\end{align*}
	Since $x_k$	is a maximum point of $g$, we have $\Delta g(x_k)\leq 0$. Using the expression for $\Delta g$, the identity $\nabla g(x_k)=0$, and
	\begin{align*}
		\langle \nabla f,\nabla r\rangle(x_k)
		=
		\frac{2u(x_k)r(x_k)}{k(r^2(x_k)+2)\log(r^2(x_k)+2)}
		\ ,
	\end{align*}
	which we have derived from \eqref{eq:boundprin9}, we obtain
	\begin{align*}
		\Delta f(x_k)
		&\le
		\frac{2u(x_k)\bigl(1+r(x_k)\Delta r(x_k)\bigr)}{k(r^2(x_k)+2)\log(r^2(x_k)+2)}
		-
		\frac{4u(x_k)r^2(x_k)}{k(r^2(x_k)+2)^2\log(r^2(x_k)+2)}\\
		&\qquad
		-
		\frac{4(k-1)u(x_k)r^2(x_k)}{k^2(r^2(x_k)+2)^2\log^2(r^2(x_k)+2)}
		\ .
	\end{align*}
	In particular, by discarding the last two nonpositive terms, we get
	\begin{align}\label{eq:boundprin10}
		\Delta f(x_k)
		\le
		\frac{2u(x_k)\bigl(1+r(x_k)\Delta r(x_k)\bigr)}{k(r^2(x_k)+2)\log(r^2(x_k)+2)}
		\ .
	\end{align}
	First, it is easy to see that the following real function is bounded by a constant $C>0$
	\begin{align}\label{eq:boundprin20}
		\frac{1}{(x^2+2)\log(x^2+2)} \leq C
	\ \ . 
	\end{align}
	Then it follows from \eqref{eq:boundprin10}
	\begin{align}
		\Delta f(x_k) 
		&\leq \frac{2u(x_k)\bigl(1+r(x_k)\Delta r(x_k)\bigr)}{k(r^2(x_k)+2)\log(r^2(x_k)+2)}\notag \\
		\commb{$u\leq A$}\hspace*{5mm}&\leq\frac{2A}{k} \left( \frac{1}{(r^2(x_k)+2)\log(r^2(x_k)+2)} + \frac{r(x_k)\Delta r(x_k)}{(r^2(x_k)+2)\log(r^2(x_k)+2)}\right)\notag\\
		\commb{\eqref{eq:boundprin14}, \eqref{eq:boundprin20}}\hspace*{5mm}&\leq \frac{2A}{k} \left( C +2\Delta r(x_k)   \right) \label{eq:boundprin11}
	\ \ .
	\end{align}
	We now apply the estimate of Lemma~\ref{lm:distest}. We can do this because under the assumption that $r(x_k)\to\infty$ we have $r(x_k)\geq 1$ for $k$ sufficienly large, so that
	\begin{align*}
		\Delta r(x)\leq K(x)
		\ .
	\end{align*}
	But, as we already observed, $K(x)$ is uniformly bounded from above when $r(x) \geq 1$ since the Ricci curvature is bounded from below.\\
	So we conclude in \eqref{eq:boundprin11} that there is a constant $\tilde C$ for which
	\begin{align*}
		\Delta f(x_k)\leq \frac{\tilde C}{k}
		\ .
	\end{align*}

\end{proof}

\paragraph{A boundedness result}\mbox{}\\
We will also use this theorem to obtain a boundedness result for a certain differential inequality. We first make a few preliminary observations, and then state the theorem at the end of the section.\\

Let $M$ be a complete Riemannian manifold with Ricci curvature bounded from below, and let $u\in C^2(M)$ satisfy
\begin{equation}\label{eq:boundprin2}
	\Delta u \geq f(u)\ \ , 
\end{equation}
where the assumptions on $f$ will be specified later.\\
Now let $F\in C^2(\R)$ be a positive function, whose precise form will also be given later, which we require to be strictly increasing or in other terms to have a strictly positive derivative.
Since $F>0$, the function $-1/F$ is bounded from above. We may therefore apply Theorem \ref{th:maxprin} to $-1/F(u)$ and obtain a sequence $(x_k)_k$ such that
\begin{equation}\label{eq:boundprin7}
	\lim_{k\to \infty} \frac{1}{F(u(x_k))} = \inf \frac{1}{F(u)}
	= -\sup\left(-\frac{1}{F(u)}\right).
\end{equation}
Moreover, the corresponding gradient and Laplacian estimates hold:
\begin{align}
	\frac{|F'(u(x_k))|\,|\nabla u|(x_k)}{F(u(x_k))^2}\le\frac{1}{k}, \label{eq:boundprin3}
\end{align}
\begin{align}\label{eq:boundprin16}
	-\frac{F''(u(x_k))\,|\nabla u|^2(x_k)}{F(u(x_k))^2}
	-\frac{F'(u(x_k))\,\Delta u(x_k)}{F(u(x_k))^2}
	+\frac{2(F'(u(x_k)))^2\,|\nabla u|^2(x_k)}{F(u(x_k))^3}
	\ge -\frac{1}{k}\ .
\end{align}
For simplicity, from now on we evaluate all quantities at $x_k$.\\
Then, multiplying \eqref{eq:boundprin16} by $\displaystyle \frac{F'(u)^2}{F(u)^2|F''(u)|}$, we obtain
\begin{multline}\label{eq:boundprin1}
	-\ \frac{F''(u)}{|F''(u)|} \frac{F'(u)^2|\nabla u|^2}{F(u)^4}
	\ -\ \frac{F'(u)^3}{F(u)^4}\frac{\Delta u}{|F''(u)|}\\
	\ +\ \frac{2F'(u)^2}{F(u)|F''(u)|} \frac{F'(u)^2|\nabla u|^2}{F(u)^4}
	\ge -\frac{F'(u)^2}{kF(u)^2|F''(u)|}\ \ .
\end{multline}
We now estimate the terms on the left-hand side of \eqref{eq:boundprin1}. First,
\begin{align*}
	-\frac{F'(u)^3}{F(u)^4}\frac{\Delta u}{|F''(u)|}
	\leq
	-\frac{F'(u)^3}{F(u)^4}\frac{f(u)}{|F''(u)|}
	\ \ . && \commb{by \eqref{eq:boundprin2}, $F'>0$}
\end{align*}
Moreover,
\begin{align*}
	-\ \frac{F''(u)}{|F''(u)|} \frac{F'(u)^2|\nabla u|^2}{F(u)^4}
	\leq
	\left| \frac{F'(u)^2|\nabla u|^2}{F(u)^4} \right|
	\leq \frac{1}{k^2}
	\ \ . && \commb{by \eqref{eq:boundprin3}}
\end{align*}
Also,
\begin{align*}
	\frac{2F'(u)^2}{F(u)|F''(u)|} \frac{F'(u)^2|\nabla u|^2}{F(u)^4}
	&\leq
	2 \left| \frac{F'(u)^2}{F(u)|F''(u)|} \right|
	\left| \frac{F'(u)^2 |\nabla u|^2}{F(u)^4} \right|\\
	&\leq
	\frac{2}{k^2}\left| \frac{F'(u)^2}{F(u)|F''(u)|} \right|
	\ \ . && \commb{by \eqref{eq:boundprin3}}
\end{align*}
Therefore, from \eqref{eq:boundprin1} we obtain
\begin{align}\label{eq:boundprin6}
	\frac{1}{k^2}
	\ -\ 
	\frac{F'(u)^3 f(u)}{F(u)^4 |F''(u)|}
	\ +\ \frac{2}{k^2} \left| \frac{F'(u)^2}{F(u)|F''(u)|} \right|
	\geq
	-\frac{1}{k}\frac{F'(u)^2}{F(u)^2|F''(u)|}
	\ \ .
\end{align}

Suppose now that $F$ satisfies
\begin{align}\label{eq:boundprin4}
	\limsup_{t\to\infty}\frac{|F'(t)|^2}{F(t)\,|F''(t)|} &< \infty
	\ \ ,
\end{align}
and
\begin{align}\label{eq:boundprin5}
	\liminf_{t\to\infty}\frac{F'(t)^3 f(t)}{F(t)^4\,|F''(t)|}>0
	\ \ .
\end{align}
We claim that this implies
\begin{align*}
	\limsup_{k\to\infty} u(x_k) < \infty
	\ \ .
\end{align*}
Indeed, suppose by contradiction that $\limsup_{k\to\infty} u(x_k)=\infty$. Passing to a subsequence if necessary we may assume that
\begin{align*}
	u(x_k)\to \infty
	\ \ .
\end{align*}
Taking $\limsup$ in \eqref{eq:boundprin6}, and using \eqref{eq:boundprin4} together with the fact that later our choice of $F$ will satisfy $F\geq 1$, we obtain
\begin{align*}
	-\liminf_{k\to\infty}
	\frac{F'(u(x_k))^3 f(u(x_k))}{F(u(x_k))^4 |F''(u(x_k))|}
	\geq 0
	\ \ ,
\end{align*}
that is,
\begin{align*}
	\liminf_{k\to\infty}
	\frac{F'(u(x_k))^3 f(u(x_k))}{F(u(x_k))^4 |F''(u(x_k))|}
	\leq 0
	\ \ ,
\end{align*}
which contradicts \eqref{eq:boundprin5}.\\

So we have proved so far that
\begin{align*}
	\limsup_{k\to\infty} u(x_k) < \infty
	\ \ .
\end{align*}
Passing to a subsequence if necessary, we may therefore assume that
\begin{align*}
	u(x_k)\to \ell
\end{align*}
for some $\ell\in\R$.\\

Let us now show that $u$ is bounded from above.\\
Since $F>0$, from \eqref{eq:boundprin7} we obtain
\begin{align*}
	\lim_{k\to\infty}\frac{1}{F(u(x_k))}
	=
	\inf \frac{1}{F(u)}
	=
	\frac{1}{\sup_M F(u)}
	\ \ ,
\end{align*}
and therefore
\begin{align*}
	F(u(x_k))\to \sup F(u)
	\ \ .
\end{align*}
On the other hand, by continuity of $F$ and the fact that $u(x_k)\to \ell$, we also have
\begin{align*}
	F(u(x_k))\to F(\ell)
	\ \ .
\end{align*}
Hence
\begin{align*}
	F(\ell)=\sup F(u)
	\ \ .
\end{align*}
Since $F$ is strictly increasing, for every $x\in M$ we have
\begin{align*}
	F(u(x))\leq F(\ell)
	\qquad \Longrightarrow \qquad
	u(x)\leq \ell
	\ \ .
\end{align*}
This means that $\ell$ is an upper bound for $u$, hence $u$ is bounded from above:
\begin{align*}
	\sup u < \infty
	\ \ .
\end{align*}

We have therefore reduced the problem to finding a function $F$ which is positive, strictly increasing, and satisfies \eqref{eq:boundprin4} and \eqref{eq:boundprin5}. Let us search for such a function of the form
\begin{align}\label{eq:boundprin15}
	F(s) = \int_a^s \left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}} dt + 1
	\ \ ,
\end{align}
where $g$ is a positive continuous function defined on $[a,\infty)$ and such that
\begin{align*}
	\int_b^\infty \left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}} dt < \infty
\end{align*}
for some $b\geq a$. If we want $F$ to be positive, the expression in \eqref{eq:boundprin15} may fail to achieve that on the whole line. So we define it on $[a,\infty)$ and then extend $F$ to all of $\R$ in such a way that it becomes a strictly increasing $C^2$ function.\\

Let us now translate conditions \eqref{eq:boundprin4} and \eqref{eq:boundprin5} into conditions on the function $g$. To do so, we first compute the relevant quantities.
\begin{align*}
	F'(s) = \left( \int_a^s g(\tau)\,d\tau \right)^{-\frac{1}{2}}
	\ \ ,
\end{align*}
and therefore
\begin{align*}
	F''(s) = -\frac{1}{2} \left( \int_a^s g(\tau)\,d\tau \right)^{-\frac{3}{2}} g(s)
	\ \ .
\end{align*}
Hence
\begin{align*}
	\frac{|F'(t)|^2}{F(t)\,|F''(t)|}
	=
	2\,\frac{\left(\int_a^t g(\tau)\,d\tau\right)^{1/2}}{F(t)\,g(t)}
	\ \ ,
\end{align*}
so condition \eqref{eq:boundprin4} is equivalent to
\begin{align*}
	\limsup_{t\to\infty}
	\frac{\left(\displaystyle\int_a^t g(\tau)\,d\tau\right)^{1/2}}{F(t)\,g(t)}
	<\infty
	\ \ .
\end{align*}
Moreover,
\begin{align*}
	\frac{F'(t)^3 f(t)}{F(t)^4 |F''(t)|}
	=
	2\,\frac{f(t)}{F(t)^4 g(t)}
	\ \ ,
\end{align*}
so condition \eqref{eq:boundprin5} is equivalent to
\begin{align*}
	\liminf_{t\to\infty}\frac{f(t)}{F(t)^4 g(t)} > 0
	\ \ .
\end{align*}

Finally, we observe that since $g$ is positive, if $s\leq t$, then
\begin{align*}
	\left( \int_a^s g(\tau)\,d\tau \right)^{-\frac{1}{2}}
	\geq
	\left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}}
	\ \ .
\end{align*}
Therefore, for $t\geq b$,
\begin{align*}
	F(t)-1
	&=
	\int_a^t \left( \int_a^s g(\tau)\,d\tau \right)^{-\frac{1}{2}} ds\\
	&\geq
	\int_b^t \left( \int_a^s g(\tau)\,d\tau \right)^{-\frac{1}{2}} ds\\
	&\geq
	\int_b^t \left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}} ds\\
	&=
	(t-b)\left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}}
	\ \ .
\end{align*}
Hence
\begin{align*}
	F(t) \geq (t-b)\left( \int_a^t g(\tau)\,d\tau \right)^{-\frac{1}{2}}
	\ \ .
\end{align*}
This allows us to further estimate the quantity appearing in \eqref{eq:boundprin4}:
\begin{align*}
	\frac{\left(\displaystyle\int_a^t g(\tau)\,d\tau\right)^{1/2}}{F(t)\,g(t)}
	&\leq
	\frac{\left(\displaystyle\int_a^t g(\tau)\,d\tau\right)^{1/2}}
	{(t-b)\left( \int_a^t g(\tau)\,d\tau \right)^{-1/2} g(t)}\\
	&=
	\frac{\displaystyle\int_a^t g(\tau)\,d\tau}{(t-b)g(t)}\\
	&\sim
	\frac{\displaystyle\int_a^t g(\tau)\,d\tau}{t\,g(t)}
	\ \ .
\end{align*}
It is convenient to rewrite what we found in terms of the primitive of $g$.Let
\[
G(t)=\int_a^t g(\tau)\,d\tau,
\]
then $G\in C^1([a,\infty))$, $G(a)=0$, and $G'(t)=g(t)>0$ for every $t\geq a$. In these terms, the previous discussion yields the following theorem.
\begin{thm}\label{th:boundprin}
	Let $M$ be a complete Riemannian manifold whose Ricci curvature is bounded from below, and let $u\in C^2(M)$ satisfy
	\[
		\Delta u \geq f(u)
	\ \ .
	\]
	A sufficient condition for $u$ to be bounded from above on $M$ is the existence of $a\in\mathbb{R}$ and of an increasing positive function $G\in C^1([a,\infty))$ such that
	\begin{align*}
		\begin{cases}
			\displaystyle \int_b^\infty G(t)^{-1/2}\,dt<\infty\text{ for some }b\geq a\\
		\displaystyle \limsup_{t\to\infty}\frac{G(t)}{t\,G'(t)}<\infty\\
		\displaystyle \liminf_{t\to\infty}\frac{f(t)}{G'(t)}>0\ .
		\end{cases}
	\end{align*}
\end{thm}
\subsection{The uniqueness theorem}

We now prove the uniqueness part of Claim \ref{cl:existenceuniquew}. We recall that, if
\[
w:\mathbb{D}\to\mathbb{R}
\]
solves the hyperbolic equation
\[
\Delta_p w=e^{2w}-q^2e^{-2w}-1,
\]
then the function
\begin{align}\label{eq:unithm6}
	\tilde w=w+\log\left(\frac{2}{1-|z|^2}\right)
\end{align}
solves an equivalent equation with respect to the Euclidean background metric. Therefore it is enough to prove uniqueness in the Euclidean formulation and then translate the conclusion back to the original one.
\begin{thm}\label{th:unithm1}
	Let $h$ be a continuous function defined on a domain $\Omega\subset \C$. Let $\tilde w$ be a solution of
	\begin{align}\label{eq:unithm}
		\Delta\tilde w = e^{2\tilde w} - h^{2} e^{-2\tilde w}
	\end{align}
	in $\Omega$, and assume that
	\begin{align*}
		\Delta\tilde w \geq 0
		\ \ ,
	\end{align*}
	and that $e^{2\tilde w}|dz|^2$ is a complete metric on $\Omega$.
	 Then any other solution $v$ of the equation satisfies $v\leq \tilde w$.
\end{thm}
\begin{proof}
	Let us define $\eta = v- \tilde w$. We need to prove that $\eta \leq 0$.\\
	We have
	\begin{align}
	\Delta \eta &= \Delta v - \Delta \tilde w\notag\\
			& \overset{\commb{\eqref{eq:unithm}}}{=} e^{2 v}- h^2 e^{-2v} -e^{2\tilde w} + h^2e^{-2\tilde w}\label{eq:unithm2}
	\ \ .
	\end{align}
	Let us now try to explicit $\tilde w$ and $\eta$ in the RHS of  \eqref{eq:unithm2},
	\begin{align*}
		e^{2v} = e^{2(\eta + \tilde w) } = e^{2\eta}e^{2\tilde w}
		\ \ .
	\end{align*}
	Let us define the quantity $|\mu|^2 = h^2 e^{-4\tilde w}$, we have then
	\begin{align*}
		h^2 e^{-2v} =h^2 e^{-2(\eta+\tilde w)}= h^2 e^{-2\tilde w}e^{-2\eta} = |\mu|^2e^{2\tilde w}e^{-2\eta},\\
		\shortintertext{and}
		h^2 e^{-2\tilde w} = \left( h^2 e^{-4\tilde w}\right)  e^{2\tilde w}		= |\mu|^2e^{2\tilde w}
	\ \ .
	\end{align*}
	So, by substituting what we just found in \eqref{eq:unithm2}, we get
	\begin{align}
		\Delta \eta &= e^{2\tilde w} e^{2\eta}-|\mu|^2 e^{2\tilde w} e^{-2\eta} -e^{2 \tilde w} + |\mu|^{2}e^{2\tilde w}\notag\\
			&= e^{2\tilde w}(e^{2\eta} -|\mu|^2 e^{-2\eta}-1+|\mu|^2)\label{eq:unithm3}
	\ \ .
	\end{align}
	The Laplacian with respect to the metric $g=e^{2\tilde w}|dz|^2$ is simply $e^{-2\tilde w}\Delta$. So in \eqref{eq:unithm3} we obtain 
	\begin{align}\label{eq:unithm5}
		\Delta_g \eta =e^{-2\tilde w}\Delta \eta = e^{2\eta}-|\mu|^2 e^{-2\eta}-1+|\mu|^2
	\ \ .
	\end{align}
	Let us now observe that $|\mu|^2\leq 1$. Indeed, by \eqref{eq:unithm},
	\begin{align*}
		e^{2\tilde w}-h^2e^{-2\tilde w} \geq 0
		\ \ ,
	\end{align*}
	hence
	\begin{align*}
		e^{2\tilde w}(1-h^2e^{-4\tilde w}) \geq 0
		\ \ ,
	\end{align*}
	that is,
	\begin{align*}
		1-|\mu|^2 \geq 0
		\ \ ,
	\end{align*}
	so $|\mu|^2\leq 1$.\\
	Therefore, from \eqref{eq:unithm5} we get
	\begin{align*}
		\Delta_g \eta
		&=
		e^{2\eta}-|\mu|^2 e^{-2\eta}-1+|\mu|^2\\
		&\geq
		e^{2\eta}-e^{-2\eta}-1
		=: f(\eta)
		\ \ .
	\end{align*}
	We now apply Theorem \ref{th:boundprin} to the function $\eta$ on the complete simply connected Riemannian surface $(\Omega,g)$. For the auxiliary function appearing in the theorem, we may take
	\begin{align*}
		\gamma(t)=t^2
	\end{align*}
	on $[1,\infty)$. A direct computation shows that $\gamma$ satisfies the required assumptions, since
	\begin{align*}
		\int_b^\infty \left(\int_1^t \tau^2\,d\tau\right)^{-1/2}dt < \infty,
		\qquad
		\limsup_{t\to\infty}\frac{\int_1^t \tau^2\,d\tau}{t^3}<\infty,
	\end{align*}
	and
	\begin{align*}
		\liminf_{t\to\infty}\frac{f(t)}{t^2}
		=
		\liminf_{t\to\infty}\frac{e^{2t}-e^{-2t}-1}{t^2}
		>0
		\ \ .
	\end{align*}
	It remains only to check that the Ricci curvature of $g$ is bounded from below which, since we are on a surface, is the same as the Gaussian curvature. 
	For the metric
	\begin{align*}
		g=e^{2\tilde w}|dz|^2
		\ \ ,
	\end{align*}
	the Gaussian curvature is
	\begin{align}\label{eq:unithm10}
		K(g)
		&=
		-e^{-2\tilde w}\Delta \tilde w\notag\\
		&=
		-e^{-2\tilde w}\bigl(e^{2\tilde w}-h^2e^{-2\tilde w}\bigr)\notag\\
		&=
		-1+h^2e^{-4\tilde w}\notag\\
		&=
		-1+|\mu|^2\geq -1
		\ \ .
	\end{align}
	Since $|\mu|^2\leq 1$, it follows that
	\begin{align*}
		-1\leq K(g)\leq 0
		\ \ ,
	\end{align*}
	and in particular, looking at \eqref{eq:unithm10}, the curvature is bounded from below. Therefore we can apply Theorem \ref{th:boundprin} and conclude that $\eta$ is bounded from above.\\
	Now we can also apply Theorem \ref{th:maxprin} to $\eta$, obtaining a sequence $(z_k)_k\subset \Omega$ such that
	\begin{align*}
		\eta(z_k)\to \bar\eta=\sup_\Omega \eta,
		\qquad
		\limsup_{k\to\infty}\Delta_g\eta(z_k)\leq 0
		\ \ .
	\end{align*}
	Since $|\mu|^2$ is continuous and bounded, by passing to a subsequence if necessary we may also assume that
	\begin{align*}
		|\mu|^2(z_k)\to a
	\end{align*}
	for some $a\leq 1$. Taking limits in \eqref{eq:unithm5}, we obtain
	\begin{align}
		0
		&\geq
		\limsup_{k\to\infty}\Delta_g\eta(z_k)\notag\\
		&=
		\lim_{k\to\infty}\left(e^{2\eta(z_k)}-|\mu|^2(z_k)e^{-2\eta(z_k)}-1+|\mu|^2(z_k)\right)\notag\\
		&=
		e^{2\bar\eta}-ae^{-2\bar\eta}-1+a\notag\\
		&=: \varphi(\bar \eta)
		\ \ . \label{eq:unithm4}
	\end{align}
	We have defined in the last expression the function $\varphi(s)=e^{2s}-ae^{-2s}-1+a$, which has strictly positive derivative $\varphi'(s)=2e^{2s}+2ae^{-2s}>0$, so that it is strictly increasing. Moreover $\varphi(0)=0$, hence \eqref{eq:unithm4} implies
	\begin{align*}
		\bar\eta\leq 0, \qquad \text{therefore, on $\Omega$,} \qquad \eta\leq 0
		\ \ .
	\end{align*}
	This finally implies $v\leq \tilde w$.
\end{proof}

\begin{cor}{}{}\label{cor:uni1}
	Let $h$ be a continuous function defined on a domain $\Omega\subset \C$ and let $\tilde w_1, \tilde w_2$ two solutions of 
	\begin{equation}
		\Delta \tilde w  = e^{2\tilde w}-h^2e^{-2\tilde w} \geq 0
	\end{equation}
	in $\Omega$ such that $e^{2\tilde w_i}$ are complete metrics on $\Omega$. Then  $\tilde w_1 \equiv \tilde w_2$.
\end{cor}
\begin{proof}
	Just observe that for the previous theorem, $\tilde w_1 \leq\tilde w_2$ and $\tilde w_2 \leq\tilde w_1$.
\end{proof}
\begin{cor}\label{cor:uni2}
	There is at most one solution of the equation \eqref{eq:box} such that the associated metric is complete.
\end{cor}
\begin{proof}
Let $w_1,w_2$ be two solutions of
\[
\Delta_p w = e^{2w} - q^2 e^{-2w} -1
\]
satisfying \eqref{eq:box2}, and assume that the metrics $e^{2w_1}ds_p^2$ and $e^{2w_2}ds_p^2$ are complete.

For $i=1,2$, define
\begin{align*}
	\tilde w_i = w_i + \log\left(\frac{2}{1-|z|^2}\right)
\ \ .
\end{align*}
Then
\[
e^{2\tilde w_i}|dz|^2 = e^{2w_i}ds_p^2,
\]
hence the metric $e^{2\tilde w_i}|dz|^2$ is complete, as it is the same metric as $e^{2w_i}ds_p^2$, assumed to be complete by hypothesis.

Now, set
\[
h(z)=\frac{4q(z)}{(1-|z|^2)^2}.
\]
By recalling the discussion in Section \ref{sc:geq} the equation can be rewritten with the Euclidean background metric so that $\tilde w_i$ solves
\[
\Delta \tilde w_i = e^{2\tilde w_i}-h^2e^{-2\tilde w_i}.
\]
Moreover, condition \eqref{eq:box2} is equivalent to
\[
e^{2\tilde w_i}-h^2e^{-2\tilde w_i}>0,
\]
and so
\[
\Delta \tilde w_i\ge 0.
\]
Therefore $\tilde w_1$ and $\tilde w_2$ satisfy the assumptions of Corollary \ref{cor:uni1}, so that
\[
\tilde w_1=\tilde w_2.
\]
Which finally implies that
\[
w_1=w_2
\ \ .
\]
\end{proof}
\section{Existence for the PDE}
\subsection{Analytic preliminaries}
In this chapter we will recall some essential concepts of functional analysis, that will be used extensively in the proof of the existence theorem. We report these results in this section as they come useful only now. 

Let us fix $\Omega\subset \R^n$ an open set. We assume the reader to be familiar with the Sobolev spaces $W^{k,p}$ and the spaces $C^{k,\alpha}$, but we will recall some basic information.\\

We will use the Sobolev immersion theorem, in the following form, so it will be useful to recall its statement.
\begin{thm}[Sobolev embedding on domains satisfying the cone condition]\label{th:sobemb}
	Let $\Omega$ be a bounded domain with regular boundary. Let $\ell\in \mathbb N$ and $1\leq p<\infty$. If
	\begin{align*}
		m<\ell-\frac{n}{p},
	\end{align*}
	then there is a continuous embedding
	\begin{align*}
		W^{\ell,p}(\Omega)\hookrightarrow C^m(\overline{\Omega}).
	\end{align*}
\end{thm}
We shall use only the following special case of the Ascoli-Arzelà theorem.
\begin{thm}[Ascoli-Arzelà]\label{th:ascoli}
	Suppose $\Omega$ to be bounded, and let $(f_j)_j$ be a sequence
	bounded in $C^{m+1}(\overline{\Omega})$. Then there exists a subsequence
	$(f_{j_k})_k$ and a function $f\in C^m(\overline{\Omega})$ such that
	\begin{align*}
		f_{j_k}\to f \qquad \text{in } C^m(\overline{\Omega}).
	\end{align*}
\end{thm}

Then we will need the following notion of convergence, which will be used show that the limit of a sequence of solutions of an equation is itself a solution.
\begin{defn}
	A sequence $(u_j)_j\subset C^2(\Omega)$
	is said to converge to $u$ in $C^2_{\loc}(\Omega)$ if for every compact set $K\subset \Omega$,
	\begin{align*}
		\|u_j-u\|_{C^2(K)}\to 0\ .
	\end{align*}
\end{defn}

Then, we will need the following theorem, whose proof can be found in \cite{gilbarg_trudinger_1983} 
\begin{thm}[Elliptic interior $L^2$-estimate]\label{th:ellintestltwo}
	Let $\Omega',\Omega\subset \mathbb R^n$ be bounded domains with
	$\overline{\Omega'}\subset \Omega$, and let $\ell\in \mathbb N\cup\{0\}$. Suppose
	that $u\in H^{\ell+2}_{\loc}(\Omega)$ satisfies
	\begin{align*}
		\Delta u = f
	\end{align*}
	on $\Omega$, with $f\in H^\ell(\Omega)$. Then $u\in H^{\ell+2}(\Omega')$ and there
	exists a constant $C=C(\ell,\Omega',\Omega)$ such that
	\begin{align*}
		\|u\|_{H^{\ell+2}(\Omega')}
		\leq C\Bigl(\|u\|_{H^1(\Omega)}+\|f\|_{H^\ell(\Omega)}\Bigr).
	\end{align*}
\end{thm}

Finally, we might recall the statement of the method for sub- and super-solutions, in the form that can be found in \cite{Wan1992}.
\begin{thm}\label{th:wan9}
	Let $M$ be a complete Riemannian manifold, and let $ F(x,u)$ be a $C^\infty$ function on $M\times \mathbb R$ such that \[ \frac{\partial F}{\partial u}>0. \]
	If there exist two functions $ \underline{\psi}\leq \overline{\psi} \in C^0(M)\cap H^1(M)$
	such that
		\begin{align*}
			\begin{cases}
				\Delta \overline{\psi} \leq F(x,\overline{\psi})\\
				\Delta \underline{\psi} \geq F(x,\underline{\psi})
			\end{cases} \text{weakly}
		\ , 
		\end{align*}
	then there is a certain $u\in C^\infty(M)$ such that
	\[
		\Delta u=F(x,u)
	\]
	and \[ \psi_-\leq u\leq \psi. \]
	We call $\underline{\psi}$ and $\overline{\psi}$ respectively the sub- and super-solutions.
\end{thm}
\subsection{The bounded case}
We may prove the existence theorem with the stronger hypothesis of boundedness for the norm of the quadratic differential. 
\begin{prop}\label{pr:exboundcase}
	Let $\Phi$ a holomorphic quadratic differential on $\mathbb{D}$, such that $\|\Phi\|$ is bounded. Then the equation \eqref{eq:box} has a unique $C^\infty$ solution $w$ that satisfies
	 \begin{align*}
		1\leq e^{2w} \leq \frac{1+\sqrt{1+4a^2}}{2}
	\ , 
	\end{align*}
	with $a=\sup\|\Phi\|$.
\end{prop}
\begin{proof}
First we observe that if
\[ F(x,u)=e^{2u}-\|\Phi\|^2(x)e^{-2u}-1, \]
then
\[
\frac{\partial F}{\partial u}(x,u)
=
2e^{2u}+2\|\Phi\|^2(x)e^{-2u}
>
0,
\]
then we can apply Theorem \ref{th:wan9} if we find a sub-solution and a super-solution.\\

First let us show that the constant function $\underline{\psi}\equiv 0$
is a subsolution.
\[
\Delta_p 0 = 0 \leq \|\Phi\|^2=e^{2\cdot 0}-\|\Phi\|^2e^{-2\cdot 0}+1
\]
Hence $0$ is a subsolution.\\

Then we may try to find another constant  $\overline{\psi}$ as a supersolution. Let us then impose $\overline{\psi}\equiv C$ and that it is a supersolution,
\begin{align*}
	0   \overset{!}{\leq} \Delta_p C-e^{2C}+\|\Phi\|^2e^{-2C}+1 
\ . 
\end{align*}
Setting $X = e^{2C}$, and setting $a=\sup \|\Phi\|$, this is equivalent to
\begin{align*}
	-X + a^2 X^{-1} +1 \leq 0
\ . 
\end{align*}
By solving the inequality we find that $X= \frac{1+\sqrt{1+4a^2} }{2}$ is a valid solution for the inequality, which results to
\begin{align*}
	\overline{\psi} = C = \frac{1}{2}\log \left( \frac{1+\sqrt{1+4a^2} }{2} \right) 
\ . 
\end{align*}
\end{proof}
\subsection{The general case}
\begin{thm}
	Let $\Phi$ be a holomorphic quadratic differential on $\mathbb{D}$, then \ref{} has a unique $C^\infty$ solution.
\end{thm}
\begin{proof}
	The idea of this proof is to construct a sequence of solutions on consecutive domains that exhaust $\mathbb{D}$ and prove their convergence in a topology sufficient to obtain the equation valid for the limit.\\

	For every positive integer $k$, denoting with $d_p(\cdot,\cdot)$ the Poincaré distance on $\mathbb{D}$, define the sequence of open bounded sets
	\begin{align*}
		D_k &:= \{ z\in \mathbb{D} \mid d_p(z,0) < k\}   \\
		&= \{ z\in \mathbb{D} \mid |z|< R_k \} 
	\ , 
	\end{align*}
	where $R_k = \frac{e^k-1}{e^k+1}<1$.
	Then
	\begin{align*}
		\overline{D_k} \subset D_{k+1}\text{ for every }k,\qquad \bigcup_{k=1}^\infty D_k = \mathbb{D}
	\ . 
	\end{align*}
	Every $D_k$ can be seen as a Riemannian manifold, with the metric 
	\[
	ds_{p,k}^2 = \frac{4R_k^2}{(R_k^2-|z|^2)^2} |dz|^2 = \lambda^2_{p,k}|dz|^2
	\ . 
	\]

	Furthermore $\overline{D_k}$ is compact and $|\Phi|$ is smooth, then $|\Phi|$ is bounded on $\overline{D_k}$.
	Hence we may apply Proposition \ref{pr:exboundcase} and find, for every $k$, a function $\tilde w_k$ that solves
	\begin{align*}
		\begin{cases}
			\Delta \tilde w_k = e^{2\tilde w_k}-|\Phi|^2 e^{-2\tilde w_k}\\
			(D_k,e^{2\tilde w_k}|dz|^2) \text{ is complete.}
		\end{cases}
	\end{align*}
	
	It may be useful to define $w_k$ as the conformal factor with respect to the metric $ds_{p,k}^2$. Then
	\begin{align*}
		\tilde  w_k &= w_k +\log \left( \lambda_{p,k} \right) \\
		&=  w_k +\log\left( \frac{2R_k}{R_k^2-|z|^2} \right) 
	\ . 
	\end{align*}
	
	In the proof of Proposition \ref{pr:exboundcase} we have seen that $e^{2w_k}\geq 1$, this translates on $\tilde w_k$ to 
	\begin{align}\label{eq:exthm2}
		e^{2\tilde w_k} = e^{2w_k}\frac{4R_k^2}{(R_k^2-|z|^2)^2}
		\geq \frac{4R_k^2}{(R_k^2-|z|^2)^2} \geq \frac{4}{(1-|z|^2)^2}
	\ , 
	\end{align}
	where in the last inequality we used that $\frac{R_k}{R_k^2-r^2}\geq \frac{1}{1-r^2}$, which follows from $0<r<R_k<1$.\\

	Now, since $(D_k,e^{2\tilde w_k}|dz|^2)$ is complete and $\tilde w_{k+1}|_{D_k}$ is still a solution to 
	\begin{align*}
		\Delta w = e^{2\tilde w}-|\Phi|^2 e^{-2\tilde w}\text{ on } D_k
	\ , 
	\end{align*}
	we can apply Theorem \ref{th:unithm1}, and obtain that
	\begin{align}\label{eq:exthm3}
		 \tilde w_{k+1} \leq \tilde w_k \text{ on } D_k
	\ . 
	\end{align}
	This implies that $(\tilde w_{k+j})_{j}$ is a sequence of pointwise decreasing functions on $D_k$. This fact in turn implies that $(\tilde w_k)_k$ is a pointwise convergent sequence of functions on $\mathbb{D}$. In fact, choose $x\in \mathbb{D}$, then there exists $k$ sufficiently large such that $x\in D_k$. From what we just showed, $(\tilde w_{k+j}(x))_j$ is convergent, hence the sequence $(\tilde w_k(x))_k$, which is well defined for $k$ sufficiently large, is convergent as well. Let $\tilde w$ be its pointwise limit.\\

	The pointwise convergence is obviously not sufficient to show that the limit satisfies the equation. For that, we will prove the convergence in $C^2_{\loc}(\mathbb{D})$ for a subsequence. Since the pointwise limit has already been found, this will be sufficient to conclude that $\tilde w$ satisfies the equation.\\

	Since we already have a sequence of domains that invades $\mathbb{D}$, we may prove the $C^2$ convergence on a certain $D_k$. Then, to get the full $C^2_{\loc}$ convergence on $\mathbb{D}$ we will apply a diagonal argument at the end. Instead, in order to have the $C^2_\loc$ convergence, we may prove the boundedness in the more regular space $C^3$ and apply the Ascoli-Arzelà theorem.\\
	
	So, let us fix a positive integer $k$ and consider the sequence of functions $(\tilde w_{k+j+2})_{j=1}^\infty$ on $D_k$. Since $D_k$ is a disk, we can choose a smooth bounded domain $B_k$ such that
	\begin{align*}
		\overline{D_k} \subset  B_k \subset \overline{B_k}\subset D_{k+1}
	\ . 
	\end{align*}
	
	We will now procede to apply a series of inequalities to $\| \tilde w_{k+j+2}\|_{C^3(\overline{D_k})}$ in order to obtain finally an estimate independent of $j$.\\

	Since $B_k$ has sufficiently regular boundary, we can apply the Sobolev immersion $H^{\ell} \hookrightarrow C^m$, with indexes $\ell = 5$, $m=3$, together with the obvious immersion $C^3(B_k)\hookrightarrow C^3(\overline{D_k})$, and obtain
	\begin{align}\label{eq:exthm1}
		\| u  \|_{C^3(\overline{D_{k}})} \leq \|u\|_{C^3(B_k)}
		\leq C(k)\|u\|_{H^5(B_k)}
	\ . 
	\end{align}
	
	We also recall the elliptic interior $L^2$-estimate: given bounded domains $\Omega',\Omega$ such that $\overline{\Omega'} \subset  \Omega$, there is a constant $C = C(\ell,\Omega',\Omega)$ such that
	\begin{align*}
		\|u\|_{H^{\ell+2}(\Omega')} \leq C \left( \|u\|_{H^1(\Omega)}+\|\Delta u\|_{H^\ell(\Omega)} \right)
	\ . 
	\end{align*}
	
	Using the equation, we will show now that we can estimate $\|\tilde w_{k+j+2}\|_{C^3(\overline{D_k})}$ using $\|\tilde w_{k+j+2}\|_{H^1(D_{k+1})}$. To simplify the notation, let us denote with $v=\tilde w_{k+j+2}$ for the remainder of the proof.\\

	We know that $v$ is a solution of the equation
	\begin{align*}
		\Delta v = e^{2v}-|\Phi|^2 e^{-2v}
	\ , 
	\end{align*}
	from which we obtain, by taking the gradient,
	\begin{align*}
		\nabla ( \Delta v ) = 2(e^{2v}+|\Phi|^2 e^{-2v})\nabla v - e^{-2v}\nabla|\Phi|^2
	\ . 
	\end{align*}
	From this we have, on $D_{k+1}$, a uniform bound in $j$ for $v=\tilde w_{k+j+2}$; in fact
	\begin{itemize}[noitemsep]
		\item[$\vartriangleright$] $|\Phi|$ and $\nabla|\Phi|^2$ are smooth, hence bounded on $\overline{D_{k+1}}$,
		\item[$\vartriangleright$] for every $j\geq 1$, from \eqref{eq:exthm3} we have 
		\begin{align*}
			v = \tilde w_{k+j+2} \leq \tilde w_{k+3}
		\qquad \text{on } D_{k+1},
		\end{align*}
		\item[$\vartriangleright$] from \eqref{eq:exthm2} we have 
		\begin{align}\label{eq:exthm4}
			e^{2v}=e^{2\tilde w_{k+j+2}} \geq \frac{4}{(1-|z|^2)^2}
		\qquad \text{on } D_{k+1}.
		\end{align}
	\end{itemize}
	Then there exists a constant $C=C(k)$ such that, pointwise on $D_{k+1}$,
	\begin{align*}
		|\nabla (\Delta v) | \leq C(k)( |\nabla v| + 1)
	\ . 
	\end{align*}
	This in turn means that
	\begin{align}\label{eq:exthm6a}
		\|\nabla ( \Delta v) \|_{L^2(D_{k+1})} \leq C(k)( \|\nabla v\|_{L^2(D_{k+1})} +1)
	\ . 
	\end{align}
	Now, again from the equation, and by applying the bounds above, we obtain
	\begin{align}\label{eq:exthm5}
		|\Delta v| = |e^{2v} -|\Phi|^2e^{-2v}| \leq C(k)\qquad \text{on }D_{k+1},
	\end{align}
	hence
	\begin{align}\label{eq:exthm5b}
		\|\Delta v\|_{L^2(D_{k+1})} \leq C(k)
	\ . 
	\end{align}
	Therefore
	\begin{align}\label{eq:exthm6}
		\|\Delta v\|_{H^1(D_{k+1})} &= \|\Delta v\|_{L^2(D_{k+1})}+\|\nabla(\Delta v)\|_{L^2(D_{k+1})}\notag\\
		&\commb{\qref{eq:exthm5b},\qref{eq:exthm6a}}\leq C(k) (1+\|\nabla v\|_{L^2(D_{k+1})}) \notag\\
		&\leq C(k) \left( 1+ \|v\|_{H^1(D_{k+1})} \right) 
	\ . 
	\end{align}

	Choose now a smooth bounded domain $\Omega'$ such that
	\begin{align*}
		\overline{B_k}\subset \Omega' \subset \overline{\Omega'} \subset D_{k+1}
	\ . 
	\end{align*}
	By the interior $L^2$-estimate with $\ell=1$, applied to the pair $(\Omega',D_{k+1})$, we obtain
	\begin{align}\label{eq:exthm11a}
		\|v\|_{H^3(\Omega')} 
		&\leq C(k,\Omega')\left( \|v\|_{H^1(D_{k+1})}+\|\Delta v\|_{H^1(D_{k+1})}\right)\notag\\
		&\commb{\qref{eq:exthm6}}\leq C(k,\Omega')\left(1+\|v\|_{H^1(D_{k+1})}\right)
	\ . 
	\end{align}
	By Sobolev immersion on $\Omega'$, from \eqref{eq:exthm11a} we also get a uniform $C^1$ bound for $v$ on $\Omega'$. Since $|\Phi|^2$ is smooth, by differentiating the equation up to order $3$ and using the previous $C^1$ bound, we obtain
	\begin{align}\label{eq:exthm11b}
		\|\Delta v\|_{H^3(\Omega')}
		\leq C(k,\Omega')\left(1+\|v\|_{H^3(\Omega')}\right)
	\ . 
	\end{align}
	Now we apply the interior $L^2$-estimate again, this time with $\ell=3$, to the pair $(B_k,\Omega')$:
	\begin{align}\label{eq:exthm11}
		\|v\|_{H^5(B_k)} 
		&\leq C(k,\Omega') \left( \|v\|_{H^1(\Omega')} + \|\Delta v\|_{H^3(\Omega')}\right)\notag\\
		&\commb{\qref{eq:exthm11b}}\leq C(k,\Omega') \left( 1+\|v\|_{H^1(\Omega')} + \|v\|_{H^3(\Omega')}\right)\notag\\
		&\commb{\qref{eq:exthm11a}}\leq C(k,\Omega') \left( 1+\|v\|_{H^1(D_{k+1})}\right)
	\ . 
	\end{align}
	So that finally it is sufficient to estimate $\|v\|_{H^1(D_{k+1})}$.

	Similarly to before, $v=\tilde w_{k+j+2}$ solves the equation on $D_{k+2}$
	\begin{align*}
		\Delta v = e^{2v}-|\Phi|^2 e^{-2v}\quad\text{ on } D_{k+2}
	\ . 
	\end{align*}
	Choose a positive test function $\varphi\in C^\infty_c(D_{k+2})$ such that $\varphi\equiv 1$ on $D_{k+1}$. Then
	\begin{align*}
		\int_{D_{k+2}} \varphi^2 v\Delta v = \int_{D_{k+2}} v \varphi^2( e^{2v}-|\Phi|^2 e^{-2v}) \geq 0
	\ . 
	\end{align*}
	This implies that
	\begin{align*}
		\int_{D_{k+2}} \nabla(\varphi^2 v)\cdot \nabla v = 
		-\int_{D_{k+2}} \varphi^2 v\Delta v \leq 0 
	\ . 
	\end{align*}
	Hence 
	\begin{align*}
		0 &\geq \int_{D_{k+2}} \nabla(\varphi^2 v)\cdot \nabla v \\
		  &= \int_{D_{k+2}} 2\varphi\,v\; \nabla \varphi\cdot  \nabla v + \varphi^2 |\nabla v|^2
	\ , 
	\end{align*}
	so that
	\begin{align*}
		\int_{D_{k+2}} \varphi^2|\nabla v|^2 
		&\leq
		- 2 \int_{D_{k+2}} \varphi v \nabla\varphi \cdot \nabla v\\
		&\leq 2\int_{D_{k+2}} \varphi v |\nabla \varphi| |\nabla v|\\
		&=  2\left<\varphi \nabla v, v \nabla \varphi \right>_{L^2(D_{k+2})}\\
		&\leq 2 \|\varphi \nabla v\|_{L^2(D_{k+2})}\| v\nabla\varphi\|_{L^2(D_{k+2})} \\
		&=  2 \left( \int_{D_{k+2}} \varphi^2 |\nabla v|^2 \right)^\frac{1}{2} \left( \int_{D_{k+2}}   v^2|\nabla \varphi|^2\right)^\frac{1}{2} \ .
	\end{align*}
	This is an inequality of the form $A^2 \leq 2AB$, where
	\begin{align*}
		A = \left( \int_{D_{k+2}} \varphi^2 |\nabla v|^2 \right)^\frac{1}{2},
		\qquad
		B = \left( \int_{D_{k+2}}   v^2|\nabla \varphi|^2\right)^\frac{1}{2}.
	\end{align*}
	If $A=0$ there is nothing to prove; otherwise, dividing by $A$, we obtain $A\leq 2B$, so that
	\begin{align}\label{eq:exthm7}
		 \int_{D_{k+2}} \varphi^2|\nabla v|^2= A^2  \leq  4B^2 =4\int_{D_{k+2}}   v^2|\nabla \varphi|^2  
	\ . 
	\end{align}
	Then, remembering that $\varphi$ is a test function that depends only on $k$ and is equal to $1$ on $D_{k+1}$,
	\begin{align*}
		\|\nabla v\|^2_{L^2(D_{k+1})}&=\int_{D_{k+1}} |\nabla v|^2 \\
								  &\leq
		\int_{D_{k+2}} \varphi^2|\nabla v|^2\\
		\commb{\qref{eq:exthm7}}\hspace{4mm}&\leq 4 \int_{D_{k+2}}v^2|\nabla \varphi|^2\\
		&\leq  4\|\nabla \varphi\|^2_{L^\infty(D_{k+2})}\| v\|^2_{L^2(D_{k+2})}\\
		&= C(k) \| v\|^2_{L^2(D_{k+2})} 
	\ . 
	\end{align*}
	Then we have
	\begin{align}\label{eq:exthm8}
		\| v\|_{H^1(D_{k+1})} &= \|v\|_{L^2(D_{k+1})} + \|\nabla v\|_{L^2(D_{k+1})}\notag\\
						   &\leq C(k) \|v\|_{L^2(D_{k+2})}
	\ . 
	\end{align}
	Since, by \eqref{eq:exthm3}, we have $v=\tilde w_{k+j+2}\leq \tilde w_{k+3}$ on $D_{k+2}$, while by \eqref{eq:exthm2} we have
	\begin{align*}
		v=\tilde w_{k+j+2}\geq \log\left( \frac{2}{1-|z|^2}\right)\qquad\text{on }D_{k+2},
	\end{align*}
	and both bounds depend only on $k$, there exists $C(k)$ such that
	\begin{align}\label{eq:exthm9}
		|v(z)|\leq C(k)\qquad \forall z\in D_{k+2}.
	\end{align}
	Hence
	\begin{align}\label{eq:exthm10}
		\|v\|_{L^2(D_{k+2})}\leq C(k).
	\end{align}
	Therefore
	\begin{align*}
		\|\tilde w_{k+j+2}\|_{C^3(\overline{D_k})}
		&\overset{\qref{eq:exthm1}}{\leq} C(k)\|\tilde w_{k+j+2}\|_{H^5(B_k)} \\
		&\overset{\qref{eq:exthm11}}{\leq} C(k)\bigl(1+\|\tilde w_{k+j+2}\|_{H^1(D_{k+1})}\bigr) \\
		&\overset{\qref{eq:exthm8}}{\leq} C(k)\bigl(1+\|\tilde w_{k+j+2}\|_{L^2(D_{k+2})}\bigr) \\
		&\overset{\qref{eq:exthm10}}{\leq} C(k).
	\end{align*}
	So the sequence $(\tilde w_{k+j+2})_j$ is uniformly bounded in $C^3(\overline{D_k})$.\\

	We may briefly illustrate the diagonal argument yielding the convergence in $C^2_\loc$.\\
	For every $a\in \N$, we proved above that the sequence $(\tilde w_{a+j+2})_{j=1}^\infty$ is uniformly bounded in $C^3(\overline{D_a})$.\\
	Define $w_{0,b}:=\tilde w_b$ and fix a positive integer index $a$. Up to taking a subsequence, we may assume that $(w_{a-1,b})_b$ is a subsequence of $(\tilde w_{a+j+2})_j$, hence it is bounded in $C^3(\overline{D_a})$. By the Ascoli-Arzelà theorem we can therefore extract a subsequence $(w_{a,b})_b$ of $(w_{a-1,b})_b$ which converges in $C^2(\overline{D_a})$.\\
	We claim that the diagonal sequence $(w_{a,a})_a$ converges in $C^2_\loc$.\\
	To show that, first fix an index $m$. By construction $(w_{m,b})_b$ converges in $C^2(\overline{D_m})$. Then, if $a_0\geq m$,
	\[
	(w_{a,a})_{a\geq a_0} \text{ is a subsequence of } (w_{m,b})_b,
	\]
	hence $(w_{a,a})_{a\geq a_0}$ converges in $C^2(\overline{D_m})$. Since $m$ is arbitrary, and every compact set $K\subset \mathbb{D}$ is contained in $D_m$ for some $m$, the diagonal sequence converges in $C^2(K)$ for every compact set $K\subset \mathbb{D}$. Denoting by $(\tilde w_{k'})$ the diagonal subsequence, we have then
	\begin{align*}
		\tilde w_{k'}\to \tilde w \qquad \text{in } C^2_\loc(\mathbb{D}),
	\end{align*}
	and since the original sequence already converges pointwise to $\tilde w$, the limit of the diagonal subsequence is necessarily the same function $\tilde w$.\\


	Now let $K\subset \mathbb{D}$ be a compact set. Since $\tilde w_{k'}\to \tilde w$ in $C^2(K)$, we have
	\begin{align*}
		\tilde w_{k'}\to \tilde w,\qquad \Delta \tilde w_{k'}\to \Delta \tilde w
	\end{align*}
	uniformly on $K$. Moreover, since $|\Phi|^2$ is fixed and smooth, from the uniform convergence of $\tilde w_{k'}$ to $\tilde w$ on $K$ we also get
	\begin{align*}
		e^{2\tilde w_{k'}}\to e^{2\tilde w},\qquad |\Phi|^2e^{-2\tilde w_{k'}}\to |\Phi|^2e^{-2\tilde w}
	\end{align*}
	uniformly on $K$. Therefore, passing to the limit in
	\begin{align*}
		\Delta \tilde w_{k'}=e^{2\tilde w_{k'}}-|\Phi|^2e^{-2\tilde w_{k'}},
	\end{align*}
	we obtain
	\begin{align*}
		\Delta \tilde w=e^{2\tilde w}-|\Phi|^2e^{-2\tilde w}
	\end{align*}
	on $K$. Since $K$ is arbitrary, the equation holds on all of $\mathbb{D}$.\\

	Finally, passing to the limit in \eqref{eq:exthm2}, we get
	\begin{align*}
		e^{2\tilde w}\geq \frac{4}{(1-|z|^2)^2}.
	\end{align*}
	Hence the metric $e^{2\tilde w}|dz|^2$ dominates the Poincaré metric on
	$\mathbb{D}$. Therefore every curve which has infinite length with respect to the
	Poincaré metric also has infinite length with respect to $e^{2\tilde w}|dz|^2$.
	Since the Poincaré metric is complete, this proves that $e^{2\tilde w}|dz|^2$ is
	complete.

	This concludes the proof.
\end{proof}
